tasks.task.result.list ↗︎

Scope: task Права на выполнение: для всех 

tasks.task.result.list(taskId)

Просмотр списка результатов к задаче.

Параметры

Параметр Описание
taskId Идентификатор задачи (int).

Пример

BX.ajax.runAction("tasks.task.result.list", {
	data: {
		taskId: 100500
	}
}).then(function (response) { console.log(response);});
