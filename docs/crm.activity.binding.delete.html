crm.activity.binding.delete ↗︎

crm.activity.binding.delete({activityId: number, entityTypeId: number, entityId: number)

Удаление привязки.

Параметры

Параметр Описание
activityIdидентификатор дела
entityTypeIdидентификатор типа элемента (Справочник доступных типов)
entityIdидентификатор элемента

В случае успеха, метод вернет true.


Удаление привязки дела возможно только для элементов, к которому у текущего пользователя есть доступ на редактирование.


Если дело привязано только к одному элементу, удалить эту привязку нельзя.


Пример

crm.activity.binding.delete?activityId=1&entityTypeId=4&entityId=1000

Успешный результат: 
{
	"result": true
}
