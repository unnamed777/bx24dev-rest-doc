landing.landing.getadditionalfields ↗︎

landing.landing.getadditionalfields(lid)

Метод для получения дополнительных полей страницы. Возвращает дополнительные поля сайта или ошибку.

Параметры

Метод Описание С версии
lid идентификатор страницы

Пример

BX24.callMethod(
	'landing.landing.getadditionalfields',
	{
		lid: 349
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
