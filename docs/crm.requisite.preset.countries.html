crm.requisite.preset.countries ↗︎

crm.requisite.preset.countries()

Возвращает возможный список стран для шаблонов реквизита.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.requisite.preset.countries",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
