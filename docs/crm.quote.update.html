crm.quote.update ↗︎

crm.quote.update(id, fields, params)

Обновляет существующее предложение.

Параметры

Параметр Описание
id Идентификатор предложения.
fields Набор полей - массив вида array("обновляемое поле"=>"значение"[, ...]), где "обновляемое поле" может принимать значения из возвращаемых методом crm.quote.fields.
Примечание: чтобы узнать требуемый формат полей, выполните метод crm.quote.fields и посмотрите формат пришедших значений этих полей.
params Набор параметров. REGISTER_HISTORY_EVENT - создать запись в истории, значение по умолчанию: "Y". Дополнительно будет отправлено уведомление ответственному за предложение.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.quote.update", 
	{ 
		id: id,
		fields: { "STATUS_ID": "SENT" }	
	}, 
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
