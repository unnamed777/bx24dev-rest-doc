crm.activity.binding.list ↗︎

crm.activity.binding.list({activityId: number)

Получить список привязок.

Параметры

Параметр Описание
activityIdидентификатор дела

Метод вернет массив, элементами которого будут массивы, содержащие:


  • entityTypeId - идентификатор типа элемента (Справочник доступных типов);
  • entityId - идентификатор элемента.

Пример

crm.activity.binding.list?activityId=1

Успешный результат: 
{
	"result": [
		{
			"entityTypeId": 1,
			"entityId": 123
		},
		{
			"entityTypeId": 2,
			"entityId": 456
		},
		{
			"entityTypeId": 3,
			"entityId": 789
		}
	]
}
