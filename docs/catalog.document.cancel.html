catalog.document.cancel ↗︎

catalog.document.cancel(id)

Метод для отмены проведения документа складского учёта по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор документа складского учёта.

Примеры

BX24.callMethod(
	'catalog.document.cancel',
	{
		id: 112
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
