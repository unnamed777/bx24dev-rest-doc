sale.propertyVariant.getFields ↗︎

Scope: sale Права на выполнение: для всех 

sale.propertyVariant.getFields()

Метод возвращает поля варианта свойства.

Возвращаемые поля:

Поле Описание
idID варианта.
nameНазвание варианта/
orderPropsIdПривязка к свойству.
sortСортировка.
valueЗначение варианта.

Параметры

Без параметров.



Примеры

BX24.callMethod(
	'sale.propertyVariant.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
