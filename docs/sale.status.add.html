sale.status.add ↗︎

Scope: sale Права на выполнение: для всех 

sale.status.add(id, type)

Метод создания статуса.

Если операция успешна, возвращается ресурс статуса в теле ответа.

Параметры

ПараметрТип Описание
idstring Код статуса.
typestring Тип статуса.


Примеры

BX24.callMethod(
	'sale.status.add', 
	{ 
		fields:{
			id: 'XX',
			type: 'D',
			notify: 'Y'
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
