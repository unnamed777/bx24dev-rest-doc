crm.vat.get ↗︎

crm.vat.get(id)

Возвращает ставку НДС по идентификатору.

Параметры

Параметр Описание
id Идентификатор ставки НДС.

Пример

	var id = prompt("Введите ID");
	BX24.callMethod(
		"crm.vat.get",
		{ "id": id },
		function(result)
		{
			if(result.error())
				console.error(result.error());
			else
				console.dir(result.data());
		}
	);
