catalog.productPropertyEnum.update ↗︎

catalog.productPropertyEnum.update(id, fields)

Метод для обновления значений списочных свойств.

Параметры

ПараметрТип Описание
idinteger Идентификатор значения списочного свойства.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.productPropertyEnum.update',
	{
		id: 122,
		fields: {
			propertyId: 128,
			value: "Средний",
			def: "Y",
			sort: 1234,
			xmlId: "M"
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
