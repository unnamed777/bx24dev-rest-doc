catalog.productPropertyEnum.getFields ↗︎

catalog.productPropertyEnum.getFields()

Метод возвращает поля значений списочного свойства.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.productPropertyEnum.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
