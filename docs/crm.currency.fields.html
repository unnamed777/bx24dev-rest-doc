crm.currency.fields ↗︎

crm.currency.fields()

Возвращает описание полей валюты.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.currency.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
