catalog.productImage.get ↗︎

catalog.productImage.get(productId, id)

Метод получает информацию по конкретному изображению в товаре или торговом предложении.

Параметры

ПараметрТип Описание
productIdstring Идентификатор товара или предложения. Обязательный параметр.
idinteger Идентификатор изображения. Обязательный параметр.

Примеры

BX24.callMethod(
	'catalog.productImage.get', 
	{
		productId: 1,
		id: 1
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
