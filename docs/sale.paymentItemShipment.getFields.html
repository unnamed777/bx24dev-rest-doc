sale.paymentItemShipment.getFields ↗︎

Scope: sale Права на выполнение: для всех 

sale.paymentItemShipment.getFields()

Метод возвращает поля привязки оплаты к отгрузке.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'sale.paymentItemShipment.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
