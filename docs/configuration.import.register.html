configuration.import.register ↗︎

Scope: configuration.import Права на выполнение: для всех 

configuration.import.register(
	data
)

Регистрация импорта.

Параметры

Параметр Описание С версии
data Массив ссылок на файлы из экспортированного архива.

Пример

$rest = CRest::call(
	'configuration.import.register',
	[
		'data' => [
			'STRUCTURE' => [],
			'FILES' => [],
			'MANIFEST' => [],
		],
	]
);
