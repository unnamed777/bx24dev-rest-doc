catalog.productImage.delete ↗︎

catalog.productImage.delete(productId, id)

Метод удаляет изображение из товара или торгового предложения.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
productIdstring Идентификатор товара или предложения. Обязательный параметр.
idinteger Идентификатор изображения. Обязательный параметр.

Примеры

BX24.callMethod(
	'catalog.productImage.delete',
	{
		productId: 1,
		id: 1
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
