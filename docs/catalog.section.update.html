catalog.section.update ↗︎

catalog.section.update(Id, fields)

Метод для обновления полей секции каталога.

Если операция успешна, возвращается ресурс секции торгового каталога в теле ответа.

Параметры

ПараметрТип Описание
Idstring номер секции торгового каталога.
fieldsobject Поля, соответствующие доступному списку полей fields .


Примеры

BX24.callMethod(
	'catalog.section.update', 
	{
		id: 13,
		fields: {
			yandexExport: Y
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
