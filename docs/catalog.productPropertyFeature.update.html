catalog.productPropertyFeature.update ↗︎

catalog.productPropertyFeature.update(id, fields)

Метод для обновления значения параметра свойства товаров или торговых предложений.

Параметры

ПараметрТип Описание
idinteger Идентификатор параметра свойства товаров или торговых предложений.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.productPropertyFeature.update',
	{
		id: 144,
		fields: {
			propertyId: 128,
			featureId: "IN_BASKET",
			moduleId: "catalog",
			isEnabled: "Y"
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
