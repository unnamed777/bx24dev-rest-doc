crm.requisite.preset.field.add ↗︎

crm.requisite.preset.field.add(preset, fields)

Добавляет поле в набор полей шаблона, связанного с реквизитом. В наборе могут быть значения из возвращаемых методом crm.requisite.preset.field.fields.

Параметры

Параметр Описание
presetПоля шаблона, к которому привязан набор.
fieldsОписание полей набора.

Пример

BX24.callMethod(
	"crm.requisite.preset.field.add",
	{
		preset:
		{
			"ID":1
		},
		fields:
		{
			"FIELD_NAME":"RQ_NAME",
			"FIELD_TITLE":"TEST",
			"IN_SHORT_LIST":"N",
			"SORT":580
		}
	},
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.info("В набор полей шаблона добавлено поле с ID " + result.data());
	}
);
