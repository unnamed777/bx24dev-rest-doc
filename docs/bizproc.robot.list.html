bizproc.robot.list ↗︎

Scope: bizproc Права на выполнение: для всех

bizproc.robot.list - возвращает список зарегистрированных приложением роботов.

Пример

BX24.callMethod(
	'bizproc.robot.list',
	{},
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
			alert("Успешно: " + result.data().join(', '));
	}
);
