catalog.storeproduct.getFields ↗︎

catalog.storeproduct.getFields()

Метод возвращает поля остатков по складам.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.storeproduct.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);

Возвращаемые поля

Поле ОписаниеТипПримечание
amount Количество.doubleТолько для чтения.
id Первичный ключ записи.integerТолько для чтения.
productId Идентификатор товара или торгового предложения.integerТолько для чтения.
quantityReserved Зарезервированное количество.doubleТолько для чтения.
storeId Идентификатор склада.integerТолько для чтения.
