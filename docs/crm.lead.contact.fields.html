crm.lead.contact.fields ↗︎

crm.lead.contact.fields()

Возвращает для связи лид-контакт описание полей, используемых методами семейства crm.lead.contact.*

Параметры

Без параметров

Пример

BX24.callMethod(
	"crm.lead.contact.fields",
	{},
	function(result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
