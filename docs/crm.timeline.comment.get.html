crm.timeline.comment.get ↗︎

crm.timeline.comment.get(id)

Возвращает информацию о комментарии.

Параметры

Параметр Описание
id Идентификатор комментария в таймлайне.

Пример:

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.timeline.comment.get ",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
