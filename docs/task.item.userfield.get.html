task.item.userfield.get ↗︎

Scope: task Права на выполнение: для всех

Метод возвращает свойство по идентификатору.

Параметры метода

ПараметрОписание
authТокен авторизации.
IDИдентификатор пользовательского поля.

Пример

$appParams = array(
	'auth' => 'q21g8vhcqmxdrbhqlbd2wh6ev1debppa',
	'ID' => 77
);
