catalog.product.offer.delete ↗︎

catalog.product.offer.delete(id)

Метод удаляет торговое предложение.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор торгового предложения.

Примеры

BX24.callMethod(
	'catalog.product.offer.delete', 
	{
		id: 81
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
