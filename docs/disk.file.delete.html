disk.file.delete ↗︎

Scope: disk Права на выполнение: для всех 

disk.file.delete

Уничтожает файл навсегда.

В ответе "result": true - успешное уничтожение файла.

Параметры

Параметр Описание
id Идентификатор файла.

Пример

BX24.callMethod(
	"disk.file.delete",
	{
		id: 10
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
