catalog.productProperty.add ↗︎

catalog.productProperty.add(fields)

Метод добавляет свойство товаров или торговых предложений.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.productProperty.add', 
	{
		fields: {
			name: "Размер",
			active: "Y",
			code: "SIZE1",
			iblockId: 16,
			propertyType: "L",
			isRequired: "N",
			listType: "L",
			filtrable: "Y",
			multiple: "N"
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
