crm.timeline.comment.fields ↗︎

Описание и пример

crm.timeline.comment.fields()

Возвращает список полей комментария таймлайна.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.timeline.comment.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);

Поля

Поле ОписаниеТипПримечание
ID Идентификатор. integer Только для чтения.
CREATED Дата добавления. datetime Только для чтения.
ENTITY_ID ID элемента, к которому привязан комментарий. integer Неизменяемое\Обязательное
ENTITY_TYPE Тип элемента, к которому привязан комментарий.
Значения:
  • lead - лид;
  • deal - сделка;
  • contact - контакт;
  • company - компания;
  • order - заказ.
 string Неизменяемое\Обязательное
AUTHOR_ID Автор. integer Неизменяемое
COMMENT Текст комментария. string
FILES Список файлов. attached_diskfile Массив значений, описанный по правилам, приведенным тут.
