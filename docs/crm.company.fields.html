crm.company.fields()

Возвращает описание полей компании, в том числе пользовательских.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.company.fields", 
	{}, 
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
