catalog.document.element.list ↗︎

catalog.document.element.list(order, filter, select, offset, limit)

Метод для получения списка товаров в документах складского учёта.

Если операция успешна, возвращается список документов в теле ответа.

Параметры

ПараметрТип Описание
orderarray Сортировка.
filterarray Фильтр.
selectarray Список получаемых полей.
offsetinteger Страница выборки, работает аналогично start.
limitinteger Размер страницы от 1 до 500 (если указан 0 или число больше максимального, то используется значение по умолчанию 50).

Примеры

BX24.callMethod(
	'catalog.document.element.list', 
	{
		'order': {
			'id': 'asc',
		},
		'filter': {
			'>id': 0,
		},
		'offset': 0,
		'limit': 50,
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.log(result.data());
	}
);

$result = CRest::call(
	'catalog.document.element.list',
	[
		'order' => [
			'id' => 'asc',
		],
		'filter' => [
			'>id' => 0,
		],
		'offset' => 0,
		'limit' => 5,
	]
	function(result)
	{
		result.next();
	}
);

echo '<pre>';
print_r($result);
echo '</pre>';
