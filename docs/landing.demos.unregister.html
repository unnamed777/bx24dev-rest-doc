landing.demos.unregister ↗︎

landing.demos.unregister(
	code
)

Метод удаляет зарегистрированный партнерский шаблон. Возвращает true или ошибку. В случае, если шаблон уже был удален, или не найден, вернет false.

Обратите внимание! Удаляется как шаблон сайта с данным кодом, так и все шаблоны страниц с данным кодом. Созданные сайты и страниц по этим шаблонам остаются нетронутыми.

Параметры

Параметр Описание С версии
code Символьный код шаблона.

Пример

BX24.callMethod(
	'landing.demos.unregister',
	{
		code: 'myfirstsite'
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
