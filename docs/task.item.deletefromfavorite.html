task.item.deletefromfavorite ↗︎

Scope: task Права на выполнение: для всех

Внимание! Метод устарел и не поддерживается. Рекомендуется использовать методы tasks.task.*.

Метод удаляет задачу из Избранного.

Параметры метода

Параметр Описание
auth Токен авторизации.
TASK_ID Идентификатор задачи.
PARAMS Параметр содержит ключ AFFECT_CHILDREN. Он указывает, добавлять ли в избранное подзадачи данной задачи.

Внимание! Соблюдение порядка следования параметров в запросе обязательно. При его нарушении запрос будет выполнен с ошибками.

Пример вызова

$request = 'http://your-domain.ru/rest/task.item.deletefromfavorite.xml?auth=mqa17fnd5cth4rpwtizyl49tbnzp7omf&TASK_ID=10&PARAMS[AFFECT_CHILDREN]=Y'
