disk.folder.rename ↗︎

Scope: disk Права на выполнение: для всех 

disk.folder.rename

Переименовывает папку.

В ответе та же структура, как и в disk.folder.get.

Параметры

Параметр Описание
id Идентификатор папки.
newName Новое имя.

Пример

BX24.callMethod(
	"disk.folder.rename",
	{
		id: 8,
		newName: 'New name'
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
