catalog.product.service.get ↗︎

catalog.product.service.get(id)

Метод для доступа к значению полей услуги по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор услуги.

Примеры

BX24.callMethod(
	'catalog.product.service.get', 
	{
		id: 222
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
