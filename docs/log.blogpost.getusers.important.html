log.blogpost.getusers.important ↗︎

Scope: log Права на выполнение: для всех

Отдает массив ID пользователей, прочитавших Важное сообщение на портале.

Параметры функции

Параметр Описание
POST_ID ID поста блога, являющегося Важным сообщением.

Пример

BX24.callMethod(
	'log.blogpost.getusers.important',
	{
		POST_ID: 345
	},
	function(result)
	{
		console.log(result.data());
	}
);

Запрос:

https://my.bitrix24.ru/rest/log.blogpost.getusers.important.json?POST_ID=345&auth=xxxxxxx

Ответ:

{"result":["1","2","3"]}
