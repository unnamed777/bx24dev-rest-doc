catalog.section.get ↗︎

catalog.section.get(id)

Метод для доступа к значению полей секции торгового каталога по ID.

Если операция успешна, возвращается ресурс секции торгового каталога в теле ответа.

Параметры

ПараметрТип Описание
idstring номер секции торгового каталога.

Примеры

BX24.callMethod(
	'catalog.section.get',
	{
		id: 13
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
