crm.productsection.update ↗︎

crm.productsection.update(id, fields)

Обновляет существующий раздел товаров.

Параметры

Параметр Описание
id Идентификатор раздела товаров.
fields Набор полей - массив вида array("обновляемое поле"=>"значение"[, ...]), где "обновляемое поле" может принимать значения из возвращаемых методом crm.productsection.fields.
Примечание: чтобы узнать требуемый формат полей, выполните метод crm.productsection.fields и посмотрите формат пришедших значений этих полей.

Пример

var id = prompt("Введите ID");
var sectionName = prompt("Введите название секции");
BX24.callMethod(
	"crm.productsection.update",
	{
		id: id,
		fields:
		{
			"NAME": sectionName
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
		{
			console.info(result.data());
		}
	}
);
