catalog.vat.getFields()

Метод возвращает поля ставки НДС.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.vat.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
