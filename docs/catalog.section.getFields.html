catalog.section.getFields ↗︎

catalog.section.getFields()

Метод возвращает поля секции торгового каталога.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.section.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
