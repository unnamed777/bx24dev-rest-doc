catalog.priceType.update ↗︎

catalog.priceType.update(id, fields)

Метод для обновления полей типа цены.

Параметры

ПараметрТип Описание
idinteger Идентификатор типа цены.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.priceType.update', 
	{
		id: 14,
		fields: {
			name: "Оптовая цена",
			base: "N",
			sort: 10,
			xmlId: "wholesale"
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
