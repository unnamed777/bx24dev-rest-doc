crm.invoice.recurring.delete ↗︎

crm.invoice.recurring.delete(id)

Удаляет существующую настройку для шаблона регулярного счета.

Параметры

Параметр Описание
idИдентификатор настройки шаблона регулярного счета.

Пример:

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.invoice.recurring.delete",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
