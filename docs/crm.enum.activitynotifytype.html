crm.enum.activitynotifytype ↗︎

crm.enum.activitynotifytype()

Возвращает элементы перечисления "Тип уведомления о начале активности" (для встреч и звонков).

Параметры

Без параметров

Пример

	BX24.callMethod(
		"crm.enum.activitynotifytype", 
		{}, 
		function(result) 
		{
			if(result.error())
				console.error(result.error());
			else
				console.dir(result.data());
		}
	);
