catalog.productPropertyFeature.getFields()

Метод возвращает поля параметра свойства товаров или торговых предложений.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.productPropertyFeature.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);

Поле ОписаниеТипПримечание

id Идентификатор параметра свойства. integer Только для чтения.
