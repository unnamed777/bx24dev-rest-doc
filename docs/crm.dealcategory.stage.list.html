crm.dealcategory.stage.list ↗︎

crm.dealcategory.stage.list(id)

Возвращает список стадий сделок для направления по идентификатору. Равносилен вызову метода crm.status.list с параметором ENTITY_ID равным результату вызова crm.dealcategory.status.

Параметры

Параметр Описание
id Идентификатор направления. Если указать id = 0 или ничего не указывать Пустые кавычки или совсем не передавать параметр. , то вернёт статусы "дефолтного" направления.
Если указать id > 0 не существующего направления Например, указываем id = 10, но направления с id=10 нет в системе. , ничего не возвращает.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.dealcategory.stage.list", 
	{ id: id }, 
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
