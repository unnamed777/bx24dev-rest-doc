crm.catalog.fields ↗︎

crm.catalog.fields()

Возвращает описание полей каталога товаров.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.catalog.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);

Поля

Поле ОписаниеТипПримечание
ID ИдентификаторintegerТолько для чтения
NAME Названиеstring
ORIGINATOR_ID Идентификатор источника данныхstringИспользуется только для привязки к внешнему источнику.
ORIGIN_ID Идентификатор элемента в источнике данныхstringИспользуется только для привязки к внешнему источнику.
XML_ID Символьный кодstringТолько для чтения
