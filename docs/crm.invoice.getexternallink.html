crm.invoice.getexternallink ↗︎

crm.invoice.getexternallink(id)

Метод возращает публичную ссылку для онлайн-счета.

Параметры

Параметр Описание
id Идентификатор счета.

Примеры

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.invoice.getexternallink",
	{ "id": id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.log(result.data());
	}
);
