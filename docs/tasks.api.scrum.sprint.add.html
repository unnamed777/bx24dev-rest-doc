tasks.api.scrum.sprint.add ↗︎

Scope: task Права на выполнение: для всех 

tasks.api.scrum.sprint.add(fields)

Метод добавляет спринт в Скрам.

Параметры

Параметр Описание
fields Поля, соответствующие доступному списку полей tasks.api.scrum.sprint.getFields.

Поле groupId обязательное.

Доступные значения для полей дат dateStart и dateEnd: ('ISO 8601', timestamp).

Доступные значения для поля status: ('active', 'planned', 'completed').

Примеры

const groupId = 1;
const name = 'Sprint 1';
const createdBy = 1;
const sort = 1;
const status = 'planned';
const dateStart = '2021-11-22T00:00:00+02:00';
const dateEnd = '2021-11-29T00:00:00+02:00';
BX24.callMethod(
	'tasks.api.scrum.sprint.add',
	{
		 fields:{
			name: name,
			groupId: groupId,
			createdBy: createdBy,
			sort: sort,
			status: status,
			dateStart: dateStart,
			dateEnd: dateEnd,
		 }
	},
	function(res)
	{
		console.log(res);
	}
);
