lists.field.get (params)

Метод возвращает данные поля. В случае успеха будет возвращен список полей с данными, иначе пустой массив.

Параметры

ПараметрОписание
IBLOCK_TYPE_IDid типа инфоблока (обязательно):
  • lists - тип инфоблока списка
  • bitrix_processes - тип инфоблока процессов
  • lists_socnet - тип инфоблока списков групп
IBLOCK_CODE/IBLOCK_IDкод или id инфоблока (обязательно)
SOCNET_GROUP_IDid группы (обязательно, если список создается для группы);
FIELD_IDID поля. (Если поле свойство инфоблока, то формат: "PROPERTY_propertyId". Если не указан в ответе будут данные по всем полям указанного списка)

Пример

var params = {
	'IBLOCK_TYPE_ID': 'lists_socnet',
	'IBLOCK_CODE': 'rest_1'
};
BX24.callMethod(
	'lists.field.get',
	params,
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
			console.log(result.data());
	}
);
