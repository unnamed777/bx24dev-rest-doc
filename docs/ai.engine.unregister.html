ai.engine.unregister ↗︎

Scope: ai_admin Права на выполнение: администратор

Метод для удаления engine.

Параметры

Параметры Описание С версии
code Код engine

В случае успеха возвращает true.

Пример

BX24.callMethod(
	'ai.engine.unregister',
	{
		code: 'ivanov_gpt',
	},
		function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
