catalog.priceTypeLang.update ↗︎

catalog.priceTypeLang.update(id, fields)

Метод для обновления перевода названия типа цен.

Параметры

ПараметрТип Описание
idinteger Идентификатор перевода названия типа цен.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.priceTypeLang.update', 
	{
		id: 537,
		fields: {			
			catalogGroupId: 14,
			lang: "ru",
			name: "Оптовая цена для партнёров"
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
