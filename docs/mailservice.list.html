mailservice.list ↗︎

mailservice.list(
)

Возвращает список всех почтовых сервисов.

Параметры

Без параметров

Пример

BX24.callMethod(
	"mailservice.list",
	{
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
