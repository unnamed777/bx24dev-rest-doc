crm.invoice.fields()

Возвращает описание полей счёта, в том числе пользовательских.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.invoice.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
