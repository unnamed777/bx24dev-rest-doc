catalog.vat.update ↗︎

catalog.vat.update(id, fields)

Метод для обновления ставки НДС.

Параметры

ПараметрТип Описание
idinteger Идентификатор ставки НДС.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.vat.update',
	{
		id: 9,
		fields: {
			name: "Налог 15%",
			rate: 15,
			sort: 20,
			active: "Y"
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
