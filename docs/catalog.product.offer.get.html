catalog.product.offer.get ↗︎

catalog.product.offer.get(id)

Метод для доступа к значению полей торгового предложения по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор торгового предложения.

Примеры

BX24.callMethod(
	'catalog.product.offer.get', 
	{
		id: 47
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
