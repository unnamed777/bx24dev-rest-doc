landing.landing.resolveIdByPublicUrl ↗︎

landing.site.resolveIdByPublicUrl(
	landingUrl,
	siteId
)

Метод по переданному относительному URL страницы возвращает идентификатор страницы.

Параметры

ПараметрОписаниеВерсия
landingUrlОтносительный URL страницы.
siteIdИдентификатор сайта.

Пример

BX24.callMethod(
	'landing.landing.resolveIdByPublicUrl',
	{
		landingUrl: '/folder/sub/folder/page/',
		siteId: 1817
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
