imopenlines.session.head.vote ↗︎

Метод для оценки диалога руководителем.

Параметры

ПараметрПримерОбязательныйОписание
SESSION_ID494ДаИдентификатор сессии.
RATING5НетКоличество звёзд от 1 до 5.
COMMENTМолодецНетКомментарий руководителя.

Обратите внимание: несмотря на то, что два параметра не обязательны, как минимум один из них должен быть заполнен.

Вызов метода и ответ

JavaScript для Битрикс24 в коробке:

BX.rest.callMethod(
	'imopenlines.session.head.vote',
	{
		SESSION_ID: 2047,
		RATING: 5,
		COMMENT: 'Text of comment'
	}
)
	.then(result => console.log(result.data()))
	.catch(result => console.warn(result.error().ex))
;

JavaScript для облачного Битрикс24:

BX24.callMethod(
	'imopenlines.session.head.vote',
	{
		CHAT_ID: 2024
	},
	function(result)
	{
		if(result.error())
		{
			console.warn(result.error().ex);
			return false;
		}

		console.log(result.data());
	}
);

Пример ответа:

true

Возможные коды ошибок

КодОписание
ACCESS_DENIEDУ текущего пользователя нет доступа к указанному чату.
CHAT_TYPEУказанный чат не является открытой линией.
CHAT_IDУказан не корректный идентификатор чата.
WRONG_PARAMETERКак минимум один из необязательных параметров должен быть указан.
