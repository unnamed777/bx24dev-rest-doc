catalog.productPropertyFeature.getAvailableFeaturesByProperty ↗︎

catalog.productPropertyFeature.getAvailableFeaturesByProperty(propertyId)

Метод для получения доступных параметров свойства товаров или торговых предложений.

Параметры

ПараметрТип Описание
propertyIdinteger Идентификатор свойства товаров или торговых предложений.

Примеры

BX24.callMethod(
	'catalog.productPropertyFeature.getAvailableFeaturesByProperty',
	{
		propertyId: 128
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
