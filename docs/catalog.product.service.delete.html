catalog.product.service.delete ↗︎

catalog.product.service.delete(id)

Метод удаляет услугу.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор услуги.

Примеры

BX24.callMethod(
	'catalog.product.service.delete', 
	{
		id: 222
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
