crm.deal.get ↗︎

crm.deal.get(id)

Возвращает сделку по идентификатору.

Параметры

Параметр Описание
id Идентификатор сделки.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.deal.get",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
