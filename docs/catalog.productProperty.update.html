catalog.productProperty.update ↗︎

catalog.productProperty.update(id, fields)

Метод для обновления полей свойства товаров или торговых предложений.

Параметры

ПараметрТип Описание
idinteger Идентификатор свойства товаров или торговых предложений.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.productProperty.update',
	{
		id: 128,
		fields: {
			isRequired: "Y",
			iblockId: 16,
			name: "Размер",
			propertyType: "L"
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
