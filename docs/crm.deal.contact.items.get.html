crm.deal.contact.items.get ↗︎

crm.deal.contact.items.get(id)

Возвращает набор контактов, связанных с указанной сделкой.

Параметры

Параметр Описание
id Идентификатор сделки.

Результат возвращается в виде массива объектов, каждый из которых содержит следующие поля:

Поле Описание
CONTACT_IDИдентификатор контакта
SORTИндекс сортировки
ROLE_IDИдентификатор роли (зарезервировано)
IS_PRIMARYФлаг первичного контакта

Пример вызова

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.deal.contact.items.get",
	{
		id: id
	},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
