catalog.extra.getFields()

Метод возвращает поля наценки

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.extra.getFields', 
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
