crm.contact.company.items.get ↗︎

crm.contact.company.items.get(id)

Возвращает набор компаний, связанных с указанным контактом.

Параметры

Параметр Описание
id Идентификатор контакта.

Результат возвращается в виде массива объектов, каждый из которых содержит следующие поля:

Поле Описание
COMPANY_IDИдентификатор компании
SORTИндекс сортировки
ROLE_IDИдентификатор роли (зарезервировано)
IS_PRIMARYФлаг первичной компании

Пример вызова

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.contact.company.items.get",
	{
		id: id 
	}, 
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
