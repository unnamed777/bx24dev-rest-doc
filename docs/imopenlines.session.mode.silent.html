imopenlines.session.mode.silent ↗︎

Метод для включения и выключения режима скрытой переписки

Параметры

ПараметрПримерОбязательныйПо умолчаниюОписание
CHAT_ID2020ДаИдентификатор чата.
ACTIVATEYНетNФлаг активации.

Вызов метода и ответ

JavaScript для Битрикс24 в коробке.

BX.rest.callMethod(
	'imopenlines.session.mode.silent',
	{
		CHAT_ID: 2047,
		ACTIVATE: 'Y'
	}
)
	.then(result => console.log(result.data()))
	.catch(result => console.warn(result.error().ex))
;

JavaScript для облачного Битрикс24:

BX24.callMethod(
	'imopenlines.session.mode.silent',
	{
		CHAT_ID: 2024
	},
	function(result)
	{
		if(result.error())
		{
			 console.warn(result.error().ex);
			return false;
		}

		console.log(result.data());
	}
);

Пример ответа:

true
