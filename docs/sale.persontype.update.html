sale.persontype.update ↗︎

Scope: sale Права на выполнение: для всех 

sale.persontype.update(id, fields)

Метод для обновления полей типа плательщика.

Если операция успешна, возвращается ресурс типа плательщика в теле ответа.

Параметры

ПараметрТип Описание
idstring Номер типа плательщика.
fieldsobject Поля, соответствующие доступному списку полей fields .


Примеры

BX24.callMethod(
	'sale.persontype.update', 
	{
		id: 101,
		fields: {
			name: 'Физическое лицо',
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
