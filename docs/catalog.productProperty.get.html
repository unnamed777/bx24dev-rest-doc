catalog.productProperty.get ↗︎

catalog.productProperty.get(id)

Метод для доступа к значению полей свойства товаров или торговых предложений по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор свойства товаров или торговых предложений.

Примеры

BX24.callMethod(
	'catalog.productProperty.get',
	{
		id: 128
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
