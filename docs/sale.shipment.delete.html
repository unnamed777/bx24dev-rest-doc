sale.shipment.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.shipment.delete(shipmentId)

Метод удаляет элемент коллекции отгрузок.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
shipmentIdstring Номер элемента коллекции отгрузок.


Примеры

BX24.callMethod(
	'sale.shipment.delete', 
	{ id: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
