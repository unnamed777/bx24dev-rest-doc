tasks.api.scrum.epic.add ↗︎

Scope: task Права на выполнение: для всех 

tasks.api.scrum.epic.add(fields)

Метод добавляет эпик в Скрам.

Параметры

Параметр Описание
fields Поля, соответствующие доступному списку полей tasks.api.scrum.epic.getFields.

Поля name и groupId обязательные.

В поле files можно передать массив значений с идентификаторами файлов, указав префикс n.

Примеры

const groupId = 1;
const name = 'Epic 1';
const description = 'Description text';
const color = '#69dafc';
const files = ['n428'];
BX24.callMethod(
	'tasks.api.scrum.epic.add',
	{
		fields:{
			name: name,
			groupId: groupId,
			color: color,
			files: files
		}
	},
	function(res)
	{
		console.log(res);
	}
);
