landing.landing.favoriteBlock ↗︎

landing.landing.favoriteBlock(
	lid,
	block,
	meta
)

Метод сохраняет имеющийся на странице блок в "Мои блоки". Возвращает идентификатор нового сохраненного блока.

Примечание: Метод может пригодиться при удалении блока из сохраненных.

Параметры

Метод Описание С версии
lidИдентификатор страницы.
blockИдентификатор блока.
metaОбъект информации для сохранения блока. Содержит поля:
  • name – название блока;
  • section – массив категорий, куда сохранить блок;
  • preview – изображение блока.

Пример

BX24.callMethod(
	'landing.landing.favoriteBlock',
	{
		lid: 11262,
		block: 81827,
		meta: {
			name: 'Мой блок',
			section: ['text', 'text_image'],
			preview: 'https://mycdn.com/pic/1.jpg'
		}
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
