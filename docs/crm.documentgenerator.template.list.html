crm.documentgenerator.template.list ↗︎

Описание

crm.documentgenerator.template.list(select = ['*'], filter = [], order = [], start = 0)

Возвращает список шаблонов по фильтру.

Параметры

Параметр Описание
selectмассив полей для вывода. По умолчанию выводит все поля шаблона, кроме users и entityTypeId. Чтобы они появились, надо добавить дополнительно. Например, ['*', 'entityTypeId', 'users'].
orderмассив для указания порядка вывода {"id": "desc"}.
filterмассив для фильтрации.
startoffset для постраничной навигации.

Примеры фильтра

filter: {
	"code": "%_RU",
	"numeratorId": "2",
	"region": "by",
	"active": "Y"
}

В фильтр можно передать ID сущности по ключу entityTypeId. Сюда надо передавать код сделки с учетом фильтрации по направлениям. Однако, если нужно вернуть шаблоны, привязанные к любому направлению сделки, можно передать в фильтр.

filter: {
	"entityTypeId": "2%"
}

Метод вернет список шаблонов с их полями.
