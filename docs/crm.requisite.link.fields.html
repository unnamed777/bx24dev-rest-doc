crm.requisite.link.fields ↗︎

crm.requisite.link.fields()

Возвращает описание полей для связей реквизитов.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.requisite.link.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
