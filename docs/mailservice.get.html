mailservice.get ↗︎

mailservice.get(
	ID
)

Возвращает параметры указанного почтового сервиса.

Параметры

Метод Описание С версии
ID Идентификатор почтового сервиса

Пример

BX24.callMethod(
	"mailservice.get",
	{
		'ID': 10
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
