catalog.productProperty.getFields ↗︎

catalog.productProperty.getFields()

Метод возвращает поля свойства товаров или торговых предложений.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.productProperty.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);

Возвращаемые поля

Поле ОписаниеТипПримечание
active Активно ли свойство. char
code Символьный код. string
rowCount, colCount Размер поля для ввода значения (Строк х Столбцов). integer
defaultValue Значение по умолчанию. text
filtrable Выводить ли на странице списка элементов поле для фильтрации по этому свойству. char
hint Подсказка. string
iblockId Идентификатор инфоблока. integer Обязательное поле.
id Идентификатор свойства. integer Только для чтения.

linkIblockId Идентификатор инфоблока, с элементами которого связано значение. На сегодняшний день поле не используется (данное поле предназначено для типов, которые пока в REST не поддерживаются). integer
listType Внешний вид. char Только для поля типа "Список".

multipleCnt Количество полей для ввода новых множественных значений. integer

searchable Участвуют ли значения свойства в поиске. char
sort Порядок сортировки. integer
timestampX Дата последнего изменения параметров. datetime Только для чтения.
userType Пользовательский тип свойства. string
withDescription Выводить ли поле для описания значения. char
xmlId Внешний идентификатор. string
