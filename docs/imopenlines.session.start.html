imopenlines.session.start ↗︎

Метод для старта сессии.

Параметры

ПараметрПримерОбязательныйОписание
CHAT_ID494ДаИдентификатор чата.

Вызов метода и ответ

JavaScript для Битрикс24 в коробке.

BX.rest.callMethod(
	'imopenlines.session.start',
	{
		CHAT_ID: 2024
	}
)
	.then(result => console.log(result.data()))
	.catch(result => console.warn(result.error().ex))
;

JavaScript для облачного Битрикс24:

BX24.callMethod(
	'imopenlines.session.start',
	{
		CHAT_ID: 2024
	},
	function(result)
	{
		if(result.error())
		{
			console.warn(result.error().ex);
			return false;
		}

		console.log(result.data());
	}
);

Пример ответа:

true

Возможные коды ошибок

КодОписание
ACCESS_DENIEDУ текущего пользователя нет доступа к указанному чату.
CHAT_TYPEУказанный чат не является открытой линией.
CHAT_IDУказан некорректный идентификатор чата.
