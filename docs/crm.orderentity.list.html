crm.orderentity.list ↗︎

crm.orderentity.list(filter)

Метод получает список привязок заказа к сущностям по фильтру.

Если операция успешна, возвращается список элементов в теле ответа.

Параметры

ПараметрТип Описание
filterobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'crm.orderentity.list', 
	{
		filter:{
			ownerId: 9,
			ownerTypeId: 2
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
