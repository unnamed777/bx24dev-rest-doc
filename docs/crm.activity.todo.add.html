crm.activity.todo.add ↗︎

crm.activity.todo.add(
	{
		ownerTypeId: number,
		ownerId: number,
		description: string,
		deadline: DateTime,
		responsibleId: number
	}
)

Добавить дело

Параметры

Параметр Описание
ownerTypeId Идентификатор типа элемента (справочник доступных типов), к которому относится дело
ownerId Идентификатор элемента, к которому относится дело
description Текст дела
deadline Крайний срок дела
responsibleId Ответственный за дело

Результат будет содержать id - идентификатор созданного дела.

Пример

Пример использования

crm.activity.todo.add?ownerTypeId=2&ownerId=1&deadline=2022-12-31T15:00:00&description=Связаться с клиентом

Успешный результат:

{
	"result": {
		"id": 243
	}
}
