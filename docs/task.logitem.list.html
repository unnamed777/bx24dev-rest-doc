task.logitem.list ↗︎

Scope: task Права на выполнение: для всех 

task.logitem.list(id)

Возвращает историю изменений задачи.

Параметры

Параметр Описание
ID Идентификатор задачи. Обязательный параметр

Возвращаемое значение - json файл.
