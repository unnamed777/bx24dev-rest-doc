tasks.api.scrum.epic.get ↗︎

Scope: task Права на выполнение: для всех 

tasks.api.scrum.epic.get(epicId)

Метод возвращает значения полей эпика по его id.

Параметры

Параметр Описание
epicId Идентификатор эпика.

Примеры

const epicId = 1;
BX24.callMethod(
	'tasks.api.scrum.epic.get',
	{
		id: epicId,
	},
	function(res)
	{
		console.log(res);
	}
);
