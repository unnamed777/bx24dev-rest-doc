task.commentitem.isactionallowed ↗︎

Scope: task Права на выполнение: для всех

Проверяет, разрешено ли действие.

Параметры функции

ПараметрОписание
TASKID Идентификатор задачи. Обязательный параметр.
ITEMID Идентификатор комментария. Обязательный параметр.
ACTIONID Идентификатор проверяемого действия:
  • 1 - ACTION_COMMENT_ADD;
  • 2 - ACTION_COMMENT_MODIFY;
  • 3 - ACTION_COMMENT_REMOVE.
Обязательный параметр.

Внимание! Соблюдение порядка следования параметров в запросе обязательно. При его нарушении запрос будет выполнен с ошибками.

Пример

// Для комментария с ID=1205 проверяем, разрешено ли действие удаления
BX24.callMethod(
	'task.commentitem.isactionallowed',
	[13, 1205, 3],
	function(result){
		console.info(result.data());
		console.log(result);
	}
);
