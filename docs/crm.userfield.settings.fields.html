crm.userfield.settings.fields ↗︎

crm.userfield.settings.fields(type)

Возвращает описание полей настроек для типа пользовательского поля.

Параметры

Параметр Описание
type Tип пользовательского поля. Значение из списка возвращаемого методом crm.userfield.types.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.userfield.settings.fields",
	{
		type: "string"
	},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
