sale.tradeBinding.getFields ↗︎

Scope: sale Права на выполнение: для всех 

sale.tradeBinding.getFields()

Возвращает поля заказов из внешних источников.

Возвращаемые поля:

Поле Описание
idИдентификатор записи списка.
xmlIdВнешний идентификатор записи.
externalOrderIdНомер заказа во внешней системе (задается внешней системой).
orderIdНомер заказа в системе.
tradingPlatformIdНомер торговой платформы.
tradingPlatformXmlIdВнешний идентификатор торговой платформы.
paramsПоле, в котором может храниться дополнительная информация.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'sale.tradeBinding.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
