task.dependence.delete ↗︎

Scope: task Права на выполнение: для всех 

task.dependence.delete(taskIdFrom, taskIdTo, linkType)

Метод удаляет зависимость одной задачи от другой.

Параметры

Параметр Описание С версии
taskIdFrom Идентификатор задачи от которой удаляется зависимость
taskIdTo Идентификатор задачи для которой удаляется зависимость
linkType Тип зависимости: 
const LINK_TYPE_START_START =      0x00; // Связь старт-старт
const LINK_TYPE_START_FINISH =     0x01; // Связь старт-финиш
const LINK_TYPE_FINISH_START =     0x02; // Связь финиш-старт
const LINK_TYPE_FINISH_FINISH =    0x03; // Связь финиш-финиш
