crm.automation.trigger.list ↗︎

Метод для получения списка приложений и триггеров. Возвращает массив добавленных приложением триггеров с полями NAME и CODE.

Праметры

Без параметров

Пример

function getList()
{
	BX24.callMethod(
		'crm.automation.trigger.list',
		{},
		function(result)
		{
			if(result.error())
				alert("Error: " + result.error());
			else
			{
				alert("Success: data is in console");
				console.log(result.data());
			}
		}
	);
}
