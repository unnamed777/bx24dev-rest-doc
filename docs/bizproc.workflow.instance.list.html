bizproc.workflow.instance.list ↗︎

Scope: bizproc Права на выполнение: администратор

bizproc.workflow.instance.list - метод возвращает список запущенных бизнес-процессов. Алиас метода bizproc.workflow.instance, параметры и ограничения полностью совпадают.

Пример

BX24.callMethod(
	'bizproc.workflow.instance.list',
	{
		select: [
			'ID',
			'STARTED',
			'STARTED_BY',
			'TEMPLATE_ID',
			'DOCUMENT_ID'
		],
		order: {MODIFIED: 'DESC'},
		filter: {MODULE_ID: 'crm', ENTITY: 'CCrmDocumentLead'}
	},
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
		{
			console.log(result.data());
		}
	}
)
