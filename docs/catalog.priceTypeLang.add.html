catalog.priceTypeLang.add ↗︎

catalog.priceTypeLang.add(fields)

Метод добавляет перевод названия типа цен.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.priceTypeLang.add', 
	{
		fields: {
			catalogGroupId: 14,
			lang: "ru",
			name: "Оптовая цена"
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
