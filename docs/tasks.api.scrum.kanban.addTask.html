tasks.api.scrum.kanban.addTask ↗︎

Scope: task Права на выполнение: для всех 

tasks.api.scrum.kanban.addTask(sprintId, taskId, stageId)

Метод добавляет задачу в канбан Скрама.

Параметры

Параметр Описание
sprintId Идентификатор спринта. Обязательный параметр.
taskId Идентификатор задачи. Обязательный параметр.
stageId Идентификатор стадии. Обязательный параметр.

const sprintId = 1;
const taskId = 1;
const stageId = 1;

	}
