crm.currency.localizations.delete ↗︎

Удаляет выбранные локализации для валюты, указанной по символьному идентификатору.

Параметры

Параметр Описание
id Символьный идентификатор валюты.
lids Массив идентификаторов языков, локализации которых требуется удалить.

Пример

var id = prompt("Введите ID валюты");
var langId = prompt("Введите ID языка (ru, en, de)");
BX24.callMethod(
	"crm.currency.localizations.delete",
	{
		id:id,
		lids:[ langId ]
	},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
