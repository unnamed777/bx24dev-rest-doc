sale.status.get ↗︎

Scope: sale Права на выполнение: для всех 

sale.status.get(id)

Метод для доступа к полям статуса.

Если операция успешна, возвращается ресурс статуса в теле ответа.

Параметры

ПараметрТип Описание
idstring Код статуса.


Примеры

BX24.callMethod(
	'sale.status.get', 
	{ id: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
