catalog.document.element.getFields ↗︎

Описание и пример

catalog.document.element.getFields()

Метод возвращает список полей товаров документа складского учёта.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.document.element.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.log(result.data());
	}
);

Поле ОписаниеТипПримечание

docId Идентификатор документа.integerНеизменяемое поле.
elementId Идентификатор товара (catalog.product.list).integerНеизменяемое поле.
id Идентификатор товара документа.integerТолько для чтения.
purchasingPrice Закупочная цена.double
