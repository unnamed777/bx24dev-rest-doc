crm.activity.todo.updateDeadline ↗︎

crm.activity.todo.updateDeadline(
	{
		ownerTypeId: number,
		ownerId: number,
		id: number,
		value: string
	}
)

Изменить дедлайн дела

Параметры

Параметр Описание
ownerTypeId Идентификатор типа элемента (справочник доступных типов), к которому относится дело.
ownerId Идентификатор элемента, к которому относится дело.
id Идентификатор дела.
value Новый крайний срок дела.

Результат будет содержать id - идентификатор измененного дела.

Пример

Пример использования

crm.activity.todo.updateDeadline?ownerTypeId=2&ownerId=1&id=1&value=2022-12-12T15:00:00

Успешный результат:

{
	"result": {
		"id": 243
	}
}
