disk.storage.getforapp ↗︎

Scope: disk Права на выполнение: для всех 

disk.storage.getforapp

Возвращает описание хранилища, с которым может работать приложение для хранения своих данных (файлов и папок).

Возвращаемая структура аналогична приведенной в disk.storage.get.

Пример ответа:

"result": {
	"ID": "221990",
	"NAME": "bitrix.restapi",
	"CODE": null,
	"MODULE_ID": "disk",
	"ENTITY_TYPE": "restapp",
	"ENTITY_ID": "1",
	"ROOT_OBJECT_ID": "2"
}

Параметры

Без параметров.

Пример

BX24.callMethod(
	"disk.storage.getforapp",
	{},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
