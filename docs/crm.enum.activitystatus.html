crm.enum.activitystatus ↗︎

crm.enum.activitystatus()

Возвращает элементы перечисления "Статус" (STATUS).

Параметры

Без параметров

Пример

BX24.callMethod(
        "crm.enum.activitystatus", 
       {}, 
       function(result) 
       {
           if(result.error())
               console.error(result.error());
           else
               console.dir(result.data());
       }
   );
