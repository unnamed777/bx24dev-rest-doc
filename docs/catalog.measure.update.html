catalog.measure.update ↗︎

catalog.measure.update(id, fields)

Метод для обновления полей единицы измерения.

Параметры

ПараметрТип Описание
idinteger Идентификатор единицы измерения.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.measure.update', 
	{
		id: 6,
		fields: {
			measureTitle: "Kg",
			code: "kg"
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
