catalog.vat.add ↗︎

catalog.vat.add(fields)

Метод добавляет ставку НДС.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.vat.add',
	{
		fields: {
			name: "Налог 13%",
			rate: 13,
			sort: 10,
			active: "Y"
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
