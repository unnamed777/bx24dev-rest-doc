sale.paymentItemShipment.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.paymentItemShipment.delete(id)

Метод удаляет привязку оплаты к отгрузке.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор привязки оплаты к отгрузке.

Примеры

BX24.callMethod(
	'sale.paymentItemShipment.delete', 
	{
		id: 113
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
