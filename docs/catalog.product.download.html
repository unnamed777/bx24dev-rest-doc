catalog.product.download ↗︎

catalog.product.download(fileld, productId, fieldName)

Метод скачивания файлов товара торгового каталога по переданным параметрам.

Если проверка успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
fileIdstring Номер зарегистрированного файла.
productIdstring Номер товара торгового каталога.
fieldNamestring Имя поля (свойства или поля элемента информационного блока) в котором хранится файл.

Примеры

BX24.callMethod(
	'catalog.product.download', 
	{
		fields:{
			fileId: 79,
			productId: 8,
			fieldName: 'property98'
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
