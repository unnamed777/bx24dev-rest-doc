crm.activity.type.delete ↗︎

Метод для удаления подтипа дел.

Параметры

Параметр Описание С версии
TYPE_ID Идентификатор типа дела провайдера

Пример

BX24.callMethod(
	'crm.activity.type.delete',
	{
		TYPE_ID: id
	},
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
		{
			alert("Success: " + result.data());
		}
	}
);
