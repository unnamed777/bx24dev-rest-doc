sale.propertyVariant.add ↗︎

Scope: sale Права на выполнение: для всех 

sale.propertyVariant.add(fields)

Метод добавляет вариант свойства.

Если операция успешна, возвращается ресурс варианта свойства в теле ответа.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'sale.propertyVariant.add', 
	{
		fields: {
			orderPropsId: 39,
			name: 'Размер',
			value: 'M',
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
