sale.propertygroup.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.propertygroup.delete(propertyGroupsId)

Метод удаляет группу свойств.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
propertyGroupsIdstring Номер группы свойств.


Примеры

BX24.callMethod(
	'sale.propertygroup.delete', 
	{ propertyGroupsId: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
