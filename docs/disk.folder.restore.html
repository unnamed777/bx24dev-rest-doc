disk.folder.restore ↗︎

Scope: disk Права на выполнение: для всех 

disk.folder.restore

Восстанавливает папку из корзины.

В ответе та же структура, как и в disk.folder.get.

Параметры

Параметр Описание
id Идентификатор папки.

Пример

BX24.callMethod(
	"disk.folder.restore",
	{
		id: 8
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
