crm.contact.fields ↗︎

crm.contact.fields()

Возвращает описание полей контакта, в том числе пользовательских.

Параметры

Без параметров.

Пример

	BX24.callMethod(
			"crm.contact.fields", 
			{}, 
			function(result) 
			{
				if(result.error())
					console.error(result.error());
				else
					console.dir(result.data());
			}
		);
