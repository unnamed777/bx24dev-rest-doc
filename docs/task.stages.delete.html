task.stages.delete ↗︎

Scope: task Права на выполнение: для всех

Метод удаления стадии Канбана / Моего плана. Принимает на вход id стадии.

Стадия проверяется на достаточный уровень прав, а также на то, что в ней нет задач.

Параметры:

Параметр Описание С версии
isAdmin Если установлено true, то проверки прав происходить не будет. При условии, что запрашивающий является админом портала.

Возвращает true в случае успеха.
