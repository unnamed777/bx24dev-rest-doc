crm.product.property.enumeration.fields()

Возвращает описание полей элемента свойства товаров списочного типа.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.product.property.enumeration.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
