imopenlines.message.quick.save ↗︎

Метод для сохранения сообщения в быстрые ответы.

Параметры

ПараметрПримерОбязательныйОписание
CHAT_ID2014ДаИдентификатор чата.
MESSAGE_ID18715ДаИдентификатор сообщения.

Вызов метода и ответ

JavaScript

BX.rest.callMethod(
	'imopenlines.message.quick.save',
	{
		CHAT_ID: 2014,
		MESSAGE_ID: 18715
	}
)
	.then(result => console.log(result.data()))
	.catch(result => console.warn(result.error().ex))
;

Пример ответа:

true
