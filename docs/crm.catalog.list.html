crm.catalog.list ↗︎

Возвращает список товарных каталогов. Является реализацией списочного метода для товарных каталогов.

Параметры

Cм. описание списочных методов.

Пример

BX24.callMethod(
	"crm.catalog.list",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
		{
			console.dir(result.data());
			if(result.more())
				result.next();
		}
	}
);
