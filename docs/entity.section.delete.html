entity.section.delete ↗︎

Удаление раздела хранилища. Пользователь должен обладать хотя бы правами на запись (W) в хранилище.

Параметры

Параметр Описание
ENTITY Обязательный. Строковой идентификатор хранилища.
ID Обязательный. Идентификатор удаляемого раздела.

Пример

Вызов

BX24.callMethod('entity.section.delete', {ENTITY: 'menu_new', ID: 220});

Запрос

https://my.bitrix24.ru/rest/entity.section.delete.json?ENTITY=menu_new&ID=220&auth=9affe382af74d9c5caa588e28096e872

Ответ

{"result":true}
