catalog.priceTypeLang.getLanguages ↗︎

catalog.priceTypeLang.getLanguages()

Метод возвращает доступные для перевода языки.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.priceTypeLang.getLanguages', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
