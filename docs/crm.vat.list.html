crm.vat.list ↗︎

Возвращает список ставок НДС по фильтру. Является реализацией списочного метода для ставок НДС.

Параметры

См. описание списочных методов.

Пример

	BX24.callMethod(
		"crm.vat.list",
		{
			"order": { "ID": "ASC" },
			"filter": { "ACTIVE": "Y" },
			"select": [ "ID", "NAME", "RATE" ]
		},
		function(result)
		{
			if(result.error())
				console.error(result.error());
			else
			{
				console.dir(result.data());
				if(result.more())
					result.next();
			}
		}
	);
