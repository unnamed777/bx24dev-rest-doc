catalog.product.sku.update ↗︎

catalog.product.sku.update(id, fields)

Метод для обновления полей головного товара.

Параметры

ПараметрТип Описание
idinteger Идентификатор головного товара.
fieldsobject Поля, соответствующие доступному списку полей getFieldsByFilter .


Примеры

BX24.callMethod(
	'catalog.product.sku.update',
	{
		id: 8,
		fields: {
			active: "N",
			code: "t-shirt2",
			detailText: "Замечательная футболка",
			sort: 350
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
