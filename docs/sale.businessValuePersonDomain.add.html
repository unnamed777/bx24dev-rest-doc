sale.businessValuePersonDomain.add ↗︎

Scope: sale Права на выполнение: для всех 

sale.businessValuePersonDomain.add(fields.businessvaluepersondomain)

Метод добавляет элемент соответствия физическим и юридическим лицам.

Если операция успешна, возвращается ресурс элемента соответствия физическим и юридическим лицам в теле ответа.

Параметры

ПараметрТип Описание
fields.businessvaluepersondomainobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'sale.businessValuePersonDomain.add', 
	{
		fields: {
			personTypeId: 17,
			domain: E
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
