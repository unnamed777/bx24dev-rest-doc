tasks.api.scrum.sprint.update ↗︎

Scope: task Права на выполнение: для всех 

tasks.api.scrum.sprint.update(fields)

Метод изменяет спринт.

Все поля спринта доступны для обновления. Необновляемые поля можно не передавать.

Параметры

Параметр Описание
fields Поля, соответствующие доступному списку полей tasks.api.scrum.sprint.getFields.

Доступные значения для полей дат dateStart и dateEnd: ('ISO 8601', timestamp).

Доступные значения для поля status: ('active', 'planned', 'completed').

Примеры

const sprintId = 2;
const groupId = 1;
const name = 'Sprint 2';
const dateStart = '2021-11-22T00:00:00+02:00';
const dateEnd = '2021-11-29T00:00:00+02:00';
BX24.callMethod(
	'tasks.api.scrum.sprint.update',
	{
		id: sprintId,
		fields:{
			name: name,
			groupId: groupId,
			dateStart: dateStart,
			dateEnd: dateEnd,
		}
	},
	function(res)
	{
		console.log(res);
	}
);
