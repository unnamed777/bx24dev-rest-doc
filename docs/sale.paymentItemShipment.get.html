sale.paymentItemShipment.get ↗︎

Scope: sale Права на выполнение: для всех 

sale.paymentItemShipment.get(id)

Метод для доступа к значению полей привязки оплаты к отгрузке по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор привязки оплаты к отгрузке.

Примеры

BX24.callMethod(
	'sale.paymentItemShipment.get', 
	{
		id: 114
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
