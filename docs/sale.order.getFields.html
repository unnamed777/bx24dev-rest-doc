sale.order.getFields ↗︎

Scope: sale Права на выполнение: для всех

Описание и пример

sale.order.getFields()

Метод возвращает поля заказа.

Параметры

Без параметров.

Пример

BX24.callMethod(
	'sale.order.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
