catalog.document.conductList ↗︎

catalog.document.conductList(documentIds)

Метод для группового проведения документов складского учёта.

Параметры

ПараметрТип Описание
documentIdsarray Массив идентификаторов документов, которые требуется провести.

Примеры

BX24.callMethod(
	'catalog.document.conductList',
	{
		"documentIds": [
			"114",
			"112"
		]
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
