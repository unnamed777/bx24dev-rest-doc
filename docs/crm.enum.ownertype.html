crm.enum.ownertype ↗︎

Описание и пример

crm.enum.ownertype()

Метод возвращает идентификаторы типов сущностей CRM и смарт-процессов.

Параметры

Без параметров.

Пример

	BX24.callMethod(
		"crm.enum.ownertype", 
		{}, 
		function(result) 
		{
			if(result.error())
				console.error(result.error());
			else
				console.dir(result.data());
		}
	);
