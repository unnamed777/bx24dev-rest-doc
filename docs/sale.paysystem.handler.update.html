sale.paysystem.handler.update ↗︎

sale.paysystem.handler.update (
	id,
	fields
)

Метод выполняет обновление рест-обработчика.

Параметры

Параметр Описание С версии
id Идентификатор обработчика
fields Набор полей(массив вида array("поле":"значение"[, ...])), содержащий значения, описывающие обработчик. Доступные поля:
  • NAME - Название обработчика
  • CODE - Уникальный код обработчика в системе
  • SETTINGS - Настройки обработчика
  • SORT - Сортировка

Пример

var id = prompt("Введите ID");
var name = prompt("Введите имя");

BX24.callMethod('sale.paysystem.handler.update', {"id": id, "fields": {NAME : name}},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
		{
			console.info("Обработчик обновлён");
		}
	}
);
