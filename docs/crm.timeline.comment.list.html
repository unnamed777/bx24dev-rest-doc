crm.timeline.comment.list()

Возвращает список всех комментариев для определенного crm элемента.

При фильтрации обязательные поля - ENTITY_ID
Поле ОписаниеТипПримечание
ENTITY_ID ID элемента, к которому привязана запись в таймлайне. integer Неизменяемое\Обязательное
и ENTITY_TYPE
Поле ОписаниеТипПримечание
ENTITY_TYPE Тип элемента, к которому привязан комментарий.
Значения:
  • lead - лид;
  • deal - сделка;
  • contact - контакт;
  • company - компания;
  • order - заказ.
 string Неизменяемое\Обязательное
..

При выборке используйте маску "*" для выборки всех полей (без пользовательских и множественных).

Параметры

Cм. описание списочных методов.

Пример:

BX24.callMethod(
	"crm.timeline.comment.list",
	{
		filter: {
			"ENTITY_ID": 10,
			"ENTITY_TYPE": "deal",
		},
		select: [ "ID", "COMMENT", "FILES"]
	},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
		{
			console.dir(result.data());
			if(result.more())
				result.next();
		}
	}
);
