catalog.document.getFields ↗︎

Описание и пример

catalog.document.getFields()

Метод возвращает список полей документов складского учёта.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.document.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.log(result.data());
	}
);
