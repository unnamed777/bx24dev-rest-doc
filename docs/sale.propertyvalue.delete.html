sale.propertyvalue.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.propertyvalue.delete(propertyValueId)

Метод удаляет значения свойства.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
propertyValueIdstring Номер значения свойства.


Примеры

BX24.callMethod(
	'sale.propertyvalue.delete', 
	{ propertyValueId: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
