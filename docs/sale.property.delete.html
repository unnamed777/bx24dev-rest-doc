sale.property.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.property.delete(propertyId)

Метод удаляет свойство заказа.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
propertyIdstring Номер свойства заказа.


Примеры

BX24.callMethod(
	'sale.property.delete', 
	{ id: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
