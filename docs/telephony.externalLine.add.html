telephony.externalLine.add ↗︎

Метод добавляет внешнюю линию

Параметры

Параметр Описание Тип значенияС версии
NUMBER Номер внешней линии (Например: 79117654321) string
NAME Название внешней линии. Необязательный. string
CRM_AUTO_CREATE Данный параметр отвечает за создание CRM-сущностей (сделка или лид, в зависимости от режима работы CRM) при исходящих звонках из интерфейса Битрикса (например, номеронабиратель в панели чата). Принимает значение Y|N. По-умолчанию Y booleanС обновления voximplant 23.0.0

После добавления, внешнюю линию можно указать при регистрации звонка, в поле LINE_NUMBER (см telephony.externalcall.register).
