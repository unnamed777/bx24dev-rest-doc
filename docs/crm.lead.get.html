crm.lead.get ↗︎

crm.lead.get(id)

Возвращает лид по идентификатору.

Параметры

Параметр Описание
id Идентификатор лида.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.lead.get",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
