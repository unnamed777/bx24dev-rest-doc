documentgenerator.role.fillaccesses ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.role.fillaccesses(accesses = [])

Метод установит набор ролей и их привязок.



Внимание! Данный метод не обновляет существующие привязки. Он всегда перезаписывает полную карту.

Параметры

Параметр Описание
accessesМассив привязок ролей.
  • accesses[roleId] - идентификатор роли.
  • accesses[accessCode] - символьный код для привязки.


Пример

"accesses": [
	{
		"roleId": "1",
		"accessCode" : "U1", // привязать к пользователю с ИД 1
	},
	{
		"roleId": "2",
		"accessCode" : "D1", // привязать к отделу с ИД 1
	}
]
