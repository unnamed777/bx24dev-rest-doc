catalog.store.delete ↗︎

catalog.store.delete(id)

Метод для удаления склада.

Если операция успешна, возвращается Y в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор склада.

Примеры

BX24.callMethod(
	'catalog.store.delete',
	{
		id: 42,
	},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.log(result.data());
	}
);

$result = CRest::call(
	'catalog.store.delete',
	[
		'id' => 42,
	]
);

echo '<pre>';
print_r($result);
echo '</pre>';
