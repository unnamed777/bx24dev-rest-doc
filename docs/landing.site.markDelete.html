landing.site.markDelete ↗︎

landing.site.markDelete(
	id
)

Метод помечает сайт как удаленный.

Параметры

ПараметрОписаниеВерсия
idИдентификатор сайта.

Пример

BX24.callMethod(
	'landing.site.markDelete',
	{
		id: 1688
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
