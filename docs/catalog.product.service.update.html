catalog.product.service.update ↗︎

catalog.product.service.update(id, fields)

Метод для обновления полей услуги.

Параметры

ПараметрТип Описание
idinteger Идентификатор услуги.
fieldsobject Поля, соответствующие доступному списку полей getFieldsByFilter .


Примеры

BX24.callMethod(
	'catalog.product.service.update', 
	{
		id : 44,
		fields:
		{
			active: 'N',
			iblockId: 15,
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
