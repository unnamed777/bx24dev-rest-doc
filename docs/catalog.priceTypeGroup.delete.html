catalog.priceTypeGroup.delete ↗︎

catalog.priceTypeGroup.delete(id)

Метод удаляет привязку типа цен к группе покупателей.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор привязки типа цен к группе покупателей.

Примеры

BX24.callMethod(
	'catalog.priceTypeGroup.delete', 
	{
		id: 84
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
