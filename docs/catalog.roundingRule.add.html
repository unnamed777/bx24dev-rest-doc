catalog.roundingRule.add ↗︎

catalog.roundingRule.add(fields)

Метод добавляет правило округления цен.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.roundingRule.add',
	{
		fields: {
			catalogGroupId: 14,
			price: 1000,
			roundType: 4,
			roundPrecision: 100
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
