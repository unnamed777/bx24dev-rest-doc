sale.shipment.getFields ↗︎

Scope: sale Права на выполнение: для всех

Описание и пример

sale.shipment.getFields()

Метод возвращает поля отгрузки.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'sale.shipment.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
