crm.quote.fields()

Возвращает описание полей коммерческого предложения, в том числе пользовательских.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.quote.fields", 
	{}, 
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
