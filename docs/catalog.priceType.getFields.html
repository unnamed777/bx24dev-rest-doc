catalog.priceType.getFields ↗︎

catalog.priceType.getFields()

Метод возвращает поля типа цены.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.priceType.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
