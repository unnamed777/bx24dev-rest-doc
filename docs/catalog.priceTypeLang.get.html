catalog.priceTypeLang.get ↗︎

catalog.priceTypeLang.get(id)

Метод для доступа к значению полей перевода названия типа цен.

Параметры

ПараметрТип Описание
idinteger Идентификатор перевода названия типа цен.

Примеры

BX24.callMethod(
	'catalog.priceTypeLang.get', 
	{
		id: 537
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
