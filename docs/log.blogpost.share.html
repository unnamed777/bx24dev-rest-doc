log.blogpost.share ↗︎

Добавляет получателей в сообщение Ленты новостей.

Параметры функции

Параметр Описание
USER_ID ID пользователя, добавляющего новых получателей (опционально, по умолчанию - текущий пользователь, другое значение доступно только администратору в коробочной версии).
POST_ID ID сообщения Ленты новостей.
DEST Права на просмотр сообщения.
Можно использовать значения:
  • SG<X> - рабочая группа с ID=X;
  • U<X> - пользователь с ID=X;
  • DR<X> - отдел с ID=X, включая подразделы;
  • UA - все авторизованные пользователи.
