sale.paymentItemBasket.update ↗︎

Scope: sale Права на выполнение: для всех 

sale.paymentItemBasket.update(id, fields)

Метод для обновления полей привязки оплаты к корзине.

Параметры

ПараметрТип Описание
idinteger Идентификатор привязки оплаты к корзине.
fieldsobject Поля, соответствующие доступному списку полей fields.
