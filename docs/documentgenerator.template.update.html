documentgenerator.template.update ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.template.update(id, fields)

Метод обновляет существующий шаблон. Возвращает те же данные, что и при вызове documentgenerator.template.get().

Параметры

Параметр Описание
idидентификатор шаблона.
fieldsмассив полей. Аналогично методу documentgenerator.template.add, только здесь все поля необязательные.
