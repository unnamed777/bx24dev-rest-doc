crm.product.get ↗︎

crm.product.get(id)

Возвращает товар по идентификатору.

Параметры

Параметр Описание
id Идентификатор товара.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.product.get",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
