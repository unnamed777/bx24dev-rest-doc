event.offline.error ↗︎

Метод сохраняет запись в базе с пометкой об ошибке при использовании офлайновых событий. Доступность оффлайн событий можно проверить через метод feature.get.

Метод Описание Тип
process_id Идентификатор процесса, занимается обработкой записей. Обязательный параметр.
message_id Массив значений поля MESSAGE_ID записей, которые нужно пометить как ошибочные. array
