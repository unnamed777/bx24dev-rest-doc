sale.basketitem.getFields ↗︎

Scope: sale Права на выполнение: для всех

Описание и пример

sale.basketitem.getFields()

Метод возвращает поля элемента корзины.

Параметры

Без параметров.

Пример

BX24.callMethod(
	'sale.basketitem.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
