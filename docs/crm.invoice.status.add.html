crm.invoice.status.add ↗︎

С версии 19.0.0 рекомендуется использовать метод crm.status.add

crm.invoice.status.add(fields)

Создаёт новый статус счёта.

Параметры

Параметр Описание
fields Набор полей - массив вида array("поле"=>"значение"[, ...]), содержащий значения полей статуса счёта.
Примечание: чтобы узнать требуемый формат полей, выполните метод crm.invoice.status.fields и посмотрите формат пришедших значений этих полей.
