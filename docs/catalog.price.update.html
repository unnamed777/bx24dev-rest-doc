catalog.price.update ↗︎

catalog.price.update(id, fields)

Метод для обновления полей цены товара.

Параметры

ПараметрТип Описание
idinteger Идентификатор цены товара.
fieldsobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'catalog.price.update', 
	{
		id: 1,
		fields: {
		catalogGroupId: 1,
		currency: "RUB",
		price: 5000,
		producId: 1
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
