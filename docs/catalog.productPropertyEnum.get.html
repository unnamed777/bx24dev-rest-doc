catalog.productPropertyEnum.get ↗︎

catalog.productPropertyEnum.get(id)

Метод для доступа к значению списочного свойства по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор списочного свойства.

Примеры

BX24.callMethod(
	'catalog.productPropertyEnum.get',
	{
		id: 121
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
