crm.lead.fields()

Возвращает описание полей лида, в том числе пользовательских.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.lead.fields", 
	{}, 
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
