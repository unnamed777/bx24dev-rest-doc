catalog.productPropertyFeature.get ↗︎

catalog.productPropertyFeature.get(id)

Метод для доступа к значению параметра свойства товаров или торговых предложений по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор параметра свойства товаров или торговых предложений.

Примеры

BX24.callMethod(
	'catalog.productPropertyFeature.get',
	{
		id: 144
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
