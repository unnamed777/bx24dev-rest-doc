sonet_group.user.update ↗︎

Позволяет изменить роль пользователя/пользователей в рабочей группе.

Для осуществления операции текущий пользователь должен иметь права администратора соцсети.

В случае, если текущая роль пользователя - владелец группы, она не может быть изменена этим методом.

Параметры вызова

ПараметрОписание
GROUP_IDID рабочей группы.
USER_IDID пользователя (или массив ID), которому/которым меняется роль.
ROLEкод новой роли участника группы (доступны значения E - модератор и K - участник).

Пример

// Меняем роли пользователей с ID=10 и 21 в группе соцсети с ID=15 на модераторов
BX24.callMethod('sonet_group.user.update', {
	GROUP_ID: 15,
	USER_ID: [ 10, 21 ],
	ROLE: 'E'
});
