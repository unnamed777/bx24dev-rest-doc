task.item.userfield.getlist ↗︎

Scope: task Права на выполнение: для всех

Метод возвращает список свойств.

Параметры метода

ПараметрОписание
authТокен авторизации.
ORDERМассив для сортировки результата. Массив вида array("поле сортировки"=>"направление сортировки" [, ...]).
FILTERМассив фильтрации результата вида array("фильтруемое поле"=>"значение фильтра" [, ...]). Обязательный параметр.

Пример

$appParams = array(
	'auth' => 'q21g8vhcqmxdrbhqlbd2wh6ev1debppa',
	'ORDER' => array('ID' => 'asc'),
	'FILTER' => array('USER_TYPE_ID' => 'string')
);

Примеры вызова

$request = 'http://your-domain.ru/rest/task.item.userfield.getlist.xml?'.http_build_query($appParams);

	"task.item.userfield.getlist",
	{
		order:
		{
			"ID": "ASC"
		},

		}
	},
	function(result)
	{ 
	}
