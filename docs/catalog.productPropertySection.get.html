catalog.productPropertySection.get ↗︎

catalog.productPropertySection.get(propertyId)

Метод для доступа к значению секционных настроек свойства товаров или торговых предложений.

Параметры

ПараметрТип Описание
propertyIdinteger Идентификатор свойства товаров или торговых предложений.

Примеры

BX24.callMethod(
	'catalog.productPropertySection.get',
	{
		propertyId: 128
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
