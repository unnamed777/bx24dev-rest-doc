log.blogpost.update ↗︎

Scope: log Права на выполнение: для всех

Изменяет сообщение Ленты новостей.

Параметры функции

Параметр Описание
POST_ID ID сообщения Ленты новостей.
USER_ID ID автора записи (опционально, по умолчанию - текущий пользователь, другое значение доступно только администратору в коробочной версии).
POST_MESSAGE Текст сообщения.
POST_TITLE Заголовок сообщения (опционально).
DEST Права на просмотр сообщения (опционально), по умолчанию - array("UA") - всем авторизованным пользователям.
Можно использовать значения:
  • SG<X> - рабочая группа с ID=X;
  • U<X> - пользователь с ID=X;
  • DR<X> - отдел с ID=X, включая подразделы;
  • UA - все авторизованные пользователи.
SPERM Права на просмотр сообщения (опционально), по умолчанию - array("UA") - всем авторизованным пользователям.
Данный параметр поддерживается, но удобнее применять DEST.
FILES Файлы, массив значений, описанный по правилам, приведенным тут.
