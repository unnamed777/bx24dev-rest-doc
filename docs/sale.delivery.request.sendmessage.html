sale.delivery.request.sendmessage ↗︎

Scope: sale, delivery Права на выполнение: для всех

Метод для отправки сообщений о доставке.

JSON схема запроса

Пример

Запрос: 

https://my.bitrix24.ru/rest/sale.delivery.request.sendmessage

Вызов JSON: 

{
	DELIVERY_ID: 694,
	REQUEST_ID: "4757aca4931a4f029f49c0db4374d13d",
	ADDRESSEE: "MANAGER",
	MESSAGE: {
		SUBJECT: "Your order is on its way",
		BODY: "Estimated delivery price: #MONEY#",
		MONEY_VALUES: [
			351.2
		],
		STATUS: {
			MESSAGE: "Success",
			SEMANTIC: "success",
		},
	},
}

{
