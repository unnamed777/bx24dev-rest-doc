catalog.documentcontractor.getFields()

Метод возвращает массив полей с привязками CRM-поставщиков к складским документам.

Параметры

Без параметров.

Примеры

BX.rest.callMethod(
	'catalog.documentcontractor.getFields',
	{},
	function(result) {
		if (result.error())
			console.error(result.answer.error_description);
		else 
			console.log(result.data());
	}
);

Возвращаемые поля

Поле ОписаниеТипПримечание
documentId Идентификатор документа, к которому привязан поставщик. integer Обязательное поле.
entityTypeId Идентификатор типа CRM-сущности (Контакт или Компания). integer Обязательное поле.
entityId Идентификатор CRM-сущности. integer Обязательное поле.
id Идентификатор привязки сущности к документу. integer Только для чтения.
