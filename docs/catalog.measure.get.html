catalog.measure.get ↗︎

catalog.measure.get(id)

Метод для доступа к значениям полей единицы измерения по ID.

Если операция успешна, возвращается ресурс единицы измерения в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор единицы измерения.

Примеры

BX24.callMethod(
	'catalog.measure.get', 
	{
		id: 7
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
