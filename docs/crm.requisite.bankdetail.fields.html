crm.requisite.bankdetail.fields ↗︎

crm.requisite.bankdetail.fields()

Возвращает описание полей банковских реквизитов.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.requisite.bankdetail.fields",
	{},
	function(result) 
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
