catalog.catalog.getFields ↗︎

catalog.catalog.getFields()

Метод возвращает поля торгового каталога.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.catalog.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
