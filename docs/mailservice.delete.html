mailservice.delete ↗︎

mailservice.delete(
   ID
)

Удаляет почтовый сервис.

Параметры

Параметр Описание С версии
ID Идентификатор почтового сервиса

Пример

BX24.callMethod(
	"mailservice.delete",
	{
		'ID': 8
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
