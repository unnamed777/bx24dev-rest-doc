crm.product.fields()

Возвращает описание полей товара.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.product.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
