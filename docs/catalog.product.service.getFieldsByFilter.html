catalog.product.service.getFieldsByFilter ↗︎

Описание, параметры, пример

catalog.product.service.getFieldsByFilter(filter)

Метод возвращает поля услуги по фильтру.

Параметры

ПараметрТип Описание
filterobjectВ фильтр передаётся обязательное поле:
  • iblockId - тип информационного блока.


Примеры

BX24.callMethod(
	'catalog.product.service.getFieldsByFilter', 
	{
		filter: {
			iblockId: 15,
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
