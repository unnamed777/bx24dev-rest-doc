crm.invoice.delete ↗︎

crm.invoice.delete(id)

Удаляет счёт.

Параметры

Параметр Описание
id Идентификатор счета.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.invoice.delete",
	{ "id": id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
