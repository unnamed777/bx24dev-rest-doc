catalog.document.unconfirm ↗︎

Внимание! Метод устарел с версии 22.400.0. Рекомендуется использовать метод catalog.document.cancel.

catalog.document.unconfirm(id)

Метод для отмены проведения документа.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idinteger Номер документа.

Примеры

BX24.callMethod(
	'catalog.document.unconfirm',
	{
		'id': 42,
	},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.log(result.data());
	}
);

$result = CRest::call(
	'catalog.document.unconfirm',
	[
		'id' => 42,
	]
);

echo '<pre>';
print_r($result);
echo '</pre>';
