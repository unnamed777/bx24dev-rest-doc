crm.lead.details.configuration.forceCommonScopeForAll ↗︎

Метод принудительно устанавливает общую карточку лидов для всех пользователей.

Обратите внимание, что настройки карточки повторных лидов могут отличаться от настроек карточки простых лидов. Для переключения между настройками карточек лидов применяется параметр leadCustomerType.

Параметры

Параметр Описание С версии
extras Дополнительные параметры. Здесь для лидов может быть задан параметр leadCustomerType, с допустимыми значениями:
  • 1 - простые лиды,
  • 2 - повторные лиды

Пример

//---
//Установить общую карточку лидов для всех пользователей.
BX24.callMethod(
	"crm.lead.details.configuration.forceCommonScopeForAll",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
//---
