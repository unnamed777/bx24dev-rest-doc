catalog.section.add ↗︎

catalog.section.add(fields)

Метод добавляет секцию в коллекцию секций торговых каталогов.

Если операция успешна, возвращается ресурс секции торгового каталога в теле ответа.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.section.add',
	{
		fields: {
			iblockId: 5,
			iblockSectionId: 3,
			name: 'Новости',
			code: 'news'
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
