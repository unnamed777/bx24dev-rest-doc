catalog.document.element.fields ↗︎

Внимание! Метод устарел с версии 22.400.0. Рекомендуется использовать метод catalog.document.element.getFields.

Описание и пример

catalog.document.element.fields()

Метод возвращает список полей товаров документа складского учёта.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.document.element.fields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.log(result.data());
	}
);

$result = CRest::call(
	'catalog.document.element.fields'
);

echo '<pre>';
print_r($result);
echo '</pre>';
