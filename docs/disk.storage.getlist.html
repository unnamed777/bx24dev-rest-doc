disk.storage.getlist ↗︎

Scope: disk Права на выполнение: для всех 

disk.storage.getlist

Возвращает список доступных хранилищ.

Параметры

Параметр Описание
filter Необязательный параметр. Поддерживает фильтрацию по полям, которые указаны в disk.storage.getfields как USE_IN_FILTER: true.

В ответе массив объектов, структура которых аналогична disk.storage.get.

Cм. также описание списочных методов.

Пример

//поиск хранилища группы с именем содержащем "Фут"
BX24.callMethod(
	"disk.storage.getlist",
	{
		filter: {
			'ENTITY_TYPE': 'group',
			'%NAME': 'Фут'
		}
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
