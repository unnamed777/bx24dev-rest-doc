crm.persontype.fields ↗︎

crm.persontype.fields()

Возвращает описание полей для типов плательщиков.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.persontype.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
