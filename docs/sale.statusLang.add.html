sale.statusLang.add ↗︎

Scope: sale Права на выполнение: для всех 

sale.statusLang.add(fields)

Метод для добавления локализации.

Если операция успешна, возвращается ресурс локализации статусов в теле ответа.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей getFields.


Примеры

BX24.callMethod(
	'sale.statusLang.add', 
	{
		fields: {
			statusId: 'XX',
			lid: 'ru',
			name: 'Финальный'		
		}
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
