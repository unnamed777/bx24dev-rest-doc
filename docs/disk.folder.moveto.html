disk.folder.moveto ↗︎

Scope: disk Права на выполнение: для всех 

disk.folder.moveto

Перемещает папку в указанную папку.

В ответе та же структура, как и в disk.folder.get.

Параметры

Параметр Описание
id Идентификатор папки.
targetFolderId Идентификатор папки.

Пример

BX24.callMethod(
	"disk.folder.moveto",
	{
		id: 8,
		targetFolderId: 22081990
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
