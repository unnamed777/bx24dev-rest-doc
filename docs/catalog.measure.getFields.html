catalog.measure.getFields ↗︎

catalog.measure.getFields()

Метод возвращает поля единицы измерения.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.measure.getFields', 
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
