crm.activity.fields ↗︎

crm.activity.fields()

Возвращает описание полей активности.

Параметры

Без параметров

Пример

BX24.callMethod(
	"crm.activity.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
