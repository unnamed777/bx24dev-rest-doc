sale.basketitem.get ↗︎

Scope: sale Права на выполнение: для всех 

sale.basketitem.get(basketId)

Метод для доступа к полям элемент коллекции корзины и полям связанных сущностей.

Если операция успешна, возвращается ресурс корзины в теле ответа.

Параметры

ПараметрТип Описание
basketIdstring Номер элемента коллекции корзины.


Примеры

BX24.callMethod(
	'sale.basketitem.get', 
	{ id: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
