sale.delivery.extra.service.delete ↗︎

Scope: sale, delivery Права на выполнение: для всех

Метод удаляет дополнительную услугу службы доставки.

JSON схема запроса

Пример

Запрос: 

https://my.bitrix24.ru/rest/sale.delivery.extra.service.delete

{
	ID: 999
