sale.propertyVariant.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.propertyVariant.delete(id)

Метод удаляет вариант свойства.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idstring Номер варианта свойства.


Примеры

BX24.callMethod(
	'sale.propertyVariant.delete', 
	{ id: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
