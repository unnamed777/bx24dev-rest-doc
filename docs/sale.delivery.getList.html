sale.delivery.getList ↗︎

Scope: sale, delivery Права на выполнение: для всех

Метод для получения списка доступных служб доставки.

JSON схема запроса

Пример

Запрос: 

https://my.bitrix24.ru/rest/sale.delivery.getList

{
	FILTER: {
		PARENT_ID: 622
	}
