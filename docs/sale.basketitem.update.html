sale.basketitem.update ↗︎

Scope: sale Права на выполнение: для всех 

sale.basketitem.update(id, fields)

Метод для обновления полей элемента коллекции корзины.

Если операция успешна, возвращается ресурс корзины в теле ответа.

Параметры

ПараметрТип Описание
idstring Номер элемента коллекции корзины.
fieldsobject Поля, соответствующие доступному списку полей getFields .


Примеры

BX24.callMethod(
	'sale.basketitem.update', 
	{
		id: 101,
		fields: {
			quantity: 1,
			productId: 61
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
