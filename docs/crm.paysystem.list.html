crm.paysystem.list ↗︎

crm.paysystem.list(order, filter)

Возвращает список способов оплаты, применимых к предложениям либо счетам.

Параметры

Параметр Описание
orderПоля сортировки.
filterПоля фильтра.

Пример

BX24.callMethod(
	"crm.paysystem.list", {
		order: {"SORT": "ASC"},
		filter: {
			"%NAME": "Предложение",
		}
	},
	function (result)
	{
		if (result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.dir(result.data());
			if (result.more())
				result.next();
		}
	}
);
