crm.product.delete ↗︎

crm.product.delete(id)

Удаляет товар.

Параметры

Параметр Описание
id Идентификатор товара.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.product.delete",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
