ai.engine.list ↗︎

Scope: ai_admin Права на выполнение: администратор

Метод для получения списка engine.

Параметры

Без параметров

Метод без параметров возвращает список зарегистрированных engine для текущего партнера.

Пример

BX24.callMethod(
	'ai.engine.list',
	{
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
