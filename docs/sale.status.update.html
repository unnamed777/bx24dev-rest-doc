sale.status.update ↗︎

Scope: sale Права на выполнение: для всех 

sale.status.update(id)

Метод для обновления статуса.

Если операция успешна, возвращается ресурс статуса в теле ответа.

Параметры

ПараметрТип Описание
idstring Код статуса.


Примеры

BX24.callMethod(
	'sale.status.update', 
	{ 
		id: 'XX',
		fields:{
			type: 'D',
			notify: 'N'
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
