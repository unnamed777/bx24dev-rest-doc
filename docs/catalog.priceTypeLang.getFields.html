catalog.priceTypeLang.getFields ↗︎

catalog.priceTypeLang.getFields()

Метод возвращает поля перевода названия типа цен.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.priceTypeLang.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);

Возвращаемые поля

Поле ОписаниеТипПримечание
catalogGroupId ID типа цены. integer Обязательное поле.
id Идентификатор перевода названия типа цен. integer Неизменяемое поле.
lang Язык перевода. string Обязательное поле.
name Название типа цен. string Обязательное поле
