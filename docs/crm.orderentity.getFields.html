crm.orderentity.getFields()

Метод возвращает поля привязки заказа к сущности.

Возвращаемые поля

Поле ОписаниеТипПримечание
ownerId Идентификатор сущности к которой привязываем заказ. Например, ID сделки.integerНеизменяемое, обязательное поле.
orderId Идентификатор заказа.integerНеизменяемое, обязательное поле.
ownerTypeId Идентификатор типа сущности, к которой привязываем заказ. Например, если тип сущности - Сделка, то идентификатор типа сущности - 2. Узнать идентификаторы доступных для привязки заказа типов можно с помощью метода crm.enum.getOrderOwnerTypes.integerНеизменяемое, обязательное поле.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'crm.orderentity.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
