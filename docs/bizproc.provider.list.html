bizproc.provider.list ↗︎

Scope: bizproc Права на выполнение: для всех

bizproc.activity.list - возвращает список зарегистрированных приложением SMS-провайдеров.

Внимание! Методы устарели, рекомендуется к использованию Служба сообщений

Пример

BX24.callMethod(
	'bizproc.provider.list',
	{},
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
			alert("Успешно: " + result.data().join(', '));
	}
);
