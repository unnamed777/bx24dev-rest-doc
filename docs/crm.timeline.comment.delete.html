crm.timeline.comment.delete ↗︎

crm.timeline.comment.delete(id)

Удаляет комментарий.

Ограничение: если комментарий имеет привязки к нескольким элементам, необходимо сначала снять лишние привязки через crm.timeline.bindings.unbind.

Параметры

Параметр Описание
idИдентификатор комментария в таймлайне.

Пример:

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.timeline.comment.delete",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
