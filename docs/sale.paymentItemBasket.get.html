sale.paymentItemBasket.get ↗︎

Scope: sale Права на выполнение: для всех 

sale.paymentItemBasket.get(id)

Метод для доступа к значению полей привязки оплаты к корзине по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор привязки оплаты к корзине.

Примеры

BX24.callMethod(
	'sale.paymentItemBasket.get', 
	{
		id: 105
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
