access.name ↗︎

BX24.callMethod('access.name', {ACCESS:['AU']});

Получение названий прав доступа.

Параметры

Параметр Описание
ACCESS Обязательный. Список идентификаторов прав, названия для которых нужно получить.

Пример:

https://my.bitrix24.ru/rest/access.name.xml?ACCESS[]=AU&auth=d161f25928c3184678924ec127edd29a

Ответ XML 

<response>
	<result>
		<AU>
			<provider/>
			<name>Все авторизованные пользователи</name>
		</AU>
	</result>
</response>
