sale.paymentItemBasket.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.paymentItemBasket.delete(id)

Метод удаляет привязку оплаты к корзине.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор привязки оплаты к корзине.

Примеры

BX24.callMethod(
	'sale.paymentItemBasket.delete', 
	{
		id: 104
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
