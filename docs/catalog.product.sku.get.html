catalog.product.sku.get ↗︎

catalog.product.sku.get(id)

Метод для доступа к значению полей головного товара по ID.

Параметры

ПараметрТип Описание
idinteger Идентификатор головного товара.

Примеры

BX24.callMethod(
	'catalog.product.sku.get', 
	{
		id: 34
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
