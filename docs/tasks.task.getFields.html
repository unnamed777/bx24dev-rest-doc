tasks.task.getFields ↗︎

Scope: task Права на выполнение: для всех

Описание

Метод возвращает все доступные поля.

Параметры метода

Без параметров.

Примеры

BX24.callMethod(
	'tasks.task.getFields',
	{},
	function(result)
	{
		console.info(result.data());
		console.log(result);
	}
);
