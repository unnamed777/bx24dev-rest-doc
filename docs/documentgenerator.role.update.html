documentgenerator.role.update ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.role.update(id, fields)

Метод добавит новую роль. Вернет те же данные, что и при вызове на новой роли documentgenerator.role.get() на новой роли.

Параметры

Параметр Описание
idID роли.
fieldsМассив полей. Аналогично методу documentgenerator.role.add, только здесь все поля необязательные.
