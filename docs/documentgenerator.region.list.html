documentgenerator.region.list ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.region.list()

Возвращает список регионов, как установленных по умолчанию, так и пользовательских.

Параметры

Без параметров

Ответ

"regions": {
	"uk":{ // предустановленный регион
		"code": "uk",
		"title": "Великобритания",
		"languageId": "en",
	},
	"1":{ // пользовательский регион
		"id": "1",
		"title": "Турция asdf",
		"languageId":" ",
		"formatDate": "YYYY-MM-DD",
		"formatDatetime": "YYYY-MM-DD HH:MI:SS",
		"formatName": "#LAST_NAME# #NAME# #SECOND_NAME#",
	}
}
