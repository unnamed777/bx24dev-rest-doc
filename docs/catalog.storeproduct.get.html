catalog.storeproduct.get ↗︎

catalog.storeproduct.get(id)

Метод для доступа к значениям полей остатков по складам.

Параметры

ПараметрТип Описание
idinteger Первичный ключ записи.

Примеры

BX24.callMethod(
	'catalog.storeproduct.get',
	{
		id: 1
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
