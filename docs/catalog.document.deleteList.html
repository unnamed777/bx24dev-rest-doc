catalog.document.deleteList ↗︎

catalog.document.deleteList(documentIds)

Метод для группового удаления документов складского учёта.

Параметры

ПараметрТип Описание
documentIdsarray Массив идентификаторов документов, которые требуется удалить.

Примеры

BX24.callMethod(
	'catalog.document.deleteList',
	{
		"documentIds": [
			"110",
			"112"
		]
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
