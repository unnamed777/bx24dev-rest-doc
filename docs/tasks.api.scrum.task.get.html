tasks.api.scrum.task.get ↗︎

Scope: task Права на выполнение: для всех 

tasks.api.scrum.task.get(id)

Метод возвращает значения полей задачи Скрама по её id.

Параметры

Параметр Описание
id Идентификатор задачи.

Примеры

const taskId = 1;
BX24.callMethod(
	'tasks.api.scrum.task.get',
	{
		id: taskId
	},
	function(res)
	{
		console.log(res);
	}
);
