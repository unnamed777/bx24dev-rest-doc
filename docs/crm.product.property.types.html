crm.product.property.types ↗︎

crm.product.property.types()

Возвращает список типов свойств товаров.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.product.property.types",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
