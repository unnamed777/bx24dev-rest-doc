crm.userfield.enumeration.fields

Возвращает описание полей для пользовательского поля типа "enumeration" (список).

Пример

BX24.callMethod(
	"crm.userfield.enumeration.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
