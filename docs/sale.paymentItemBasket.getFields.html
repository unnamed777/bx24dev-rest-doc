sale.paymentItemBasket.getFields ↗︎

Scope: sale Права на выполнение: для всех 

sale.paymentItemBasket.getFields()

Метод возвращает поля привязки оплаты к корзине.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'sale.paymentItemBasket.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
