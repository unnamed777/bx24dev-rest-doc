crm.enum.getorderownertypes()

Метод возвращает идентификаторы типов сущностей, к которым доступна привязка заказа.

Возможные значения

Примечание: В настоящий момент привязку заказа можно осуществить только к Сделке.

"result": [
    {
     "attribute": "DYN",
     "code": "DEAL",
     "id": "2",
     "name": "Сделка"
   }
 ]

Параметры

Без параметров.

Пример

BX24.callMethod(
        "crm.enum.getorderownertypes", 
        {}, 
        function(result) 
        {
            if(result.error())
                console.error(result.error());
            else
                console.dir(result.data());
        }
    );
