crm.enum.contenttype ↗︎

crm.enum.contenttype()

Возвращает элементы перечисления "Тип содержания".

Параметры

Без параметров

Пример

	BX24.callMethod(
		"crm.enum.contenttype", 
		{}, 
		function(result) 
		{
			if(result.error())
				console.error(result.error());
			else
				console.dir(result.data());
		}
	);
