catalog.productPropertySection.list ↗︎

catalog.productPropertySection.list(select, filter, order)

Метод получает список секционных настроек свойств товаров или торговых предложений по фильтру.

Параметры

ПараметрТип Описание
selectobject Поля, соответствующие доступному списку полей fields, а также propertyId (идентификатор свойства товаров или торговых предложений).
filterobject Поля, соответствующие доступному списку полей fields, а также propertyId (идентификатор свойства товаров или торговых предложений).
orderobject Поля, соответствующие доступному списку полей fields, а также propertyId (идентификатор свойства товаров или торговых предложений).
startstring Номер страницы вывода. Работает для https запросов.


Примеры

Для JS

BX24.callMethod(
	'catalog.productPropertySection.list',
	{
		filter:{
			propertyId: 128
		},
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
		result.next();
	}
);
