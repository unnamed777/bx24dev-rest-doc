sale.shipmentproperty.delete ↗︎

Scope: sale Права на выполнение: для всех 

sale.shipmentproperty.delete(propertyId)

Метод для удаления свойства отгрузки.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
propertyIdstring Номер свойства отгрузки.


Примеры

BX24.callMethod(
	'sale.shipmentproperty.delete', 
	{ id: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
