sale.shipmentitem.getFields ↗︎

Scope: sale Права на выполнение: для всех

Описание и пример

sale.shipmentitem.getFields()

Метод возвращает поля табличной части отгрузки.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'sale.shipmentitem.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
