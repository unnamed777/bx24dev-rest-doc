documentgenerator.document.enablepublicurl ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.document.enablepublicurl(id, status = 1)

Включает / выключает публичную ссылку на документ.

Параметры

Параметр Описание
idидентификатор документа.
status1 (включить), 0 (выключить) публичную ссылку на документ.

Ответ

"publicUrl": "" // публичная ссылка
