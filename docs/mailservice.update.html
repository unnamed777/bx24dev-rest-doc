mailservice.update ↗︎

Описание

mailservice.update(
	ID,
	ACTIVE,
	NAME,
	SERVER,
	PORT,
	ENCRYPTION,
	LINK,
	SORT
)

Обновляет параметры почтового сервиса.

Параметры

Параметр Описание С версии
ID Идентификатор почтового сервиса
ACTIVE Активность сервиса (Y / N)
NAME Имя добавляемого почтового сервиса
SERVER Адрес добавляемого сервера
PORT Номер порта
ENCRYPTION Необходимость шифрования (Y / N)
LINK Ссылка на почтовый сервис
SORT Индекс сортировки

Пример

BX24.callMethod(
	"mailservice.update",
	{
		'ID': 5,
		'ACTIVE': 'N',
		'NAME': 'Почтовый сервис Yandex',
		'SERVER': 'imap.yandex.ru',
		'PORT': '993',
		'ENCRYPTION': 'Y',
		'LINK': 'https://mail.yandex.ru/',
		'SORT': '666'
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
