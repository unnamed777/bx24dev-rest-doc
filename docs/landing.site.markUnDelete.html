landing.site.markUnDelete ↗︎

landing.site.markUnDelete(
	$id
)

Метод помечает сайт как не удаленный.

Параметры

ПараметрОписаниеВерсия
IDИдентификатор страницы

Пример

BX24.callMethod(
	'landing.site.markUnDelete',
	{
		id: 1688
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
