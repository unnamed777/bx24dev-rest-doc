sale.payment.getFields ↗︎

Scope: sale Права на выполнение: для всех

Описание и пример

sale.payment.getFields()

Метод возвращает поля оплаты.

Параметры

Без параметров.

Пример

BX24.callMethod(
	'sale.payment.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
