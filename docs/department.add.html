department.add ↗︎

Метод создает подразделение. Возможно только от имени пользователя с правами изменения структуры компании.

Параметры

ПараметрОписание
NAME наименование подразделения (обязательный)
SORT параметр сортировки подразделения
PARENT идентификатор родительского подразделения.
UF_HEAD идентификатор руководителя подразделения

Вызов

BX24.callMethod('department.add', {"NAME": "Подразделение", "PARENT": 155, "UF_HEAD": 1});

Запрос

https://my.bitrix24.ru/rest/department.add.json?NAME=%D0%9F%D0%BE%D0%B4%D1%80%D0%B0%D0%B7%D0%B4%D0%B5%D0%BB%D0%B5%D0%BD%D0%B8%D0%B5&PARENT=155&UF_HEAD=1&auth=1b2234a7412080205f4318e42c7298dc
