crm.measure.list ↗︎

crm.measure.list()

Возвращает список единиц измерений.

Параметры

Cм. описание списочных методов.

Пример

BX24.callMethod(
	"crm.measure.list",
	{
		order: {"ID": "ASC"},
		filter: {"IS_DEFAULT": "Y"},
		select: ["ID", "CODE", "STAGE_ID", "SYMBOL_RUS", "SYMBOL_INTL"]
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
		{
			console.dir(result.data());
			if (result.more())
				result.next();
		}
	}
);
