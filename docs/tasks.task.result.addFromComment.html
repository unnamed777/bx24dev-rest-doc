tasks.task.result.addFromComment ↗︎

Scope: task Права на выполнение: для всех 

tasks.task.result.addFromComment(commentId)

Создание результата задачи из комментария.

Параметры

Параметр Описание
commentId Идентификатор комментария (int).

Пример

BX.ajax.runAction("tasks.task.result.addFromComment", {
	data: {
		commentId: 100500
	}
}).then(function (response) { console.log(response);});
