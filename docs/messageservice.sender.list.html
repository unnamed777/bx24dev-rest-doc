messageservice.sender.list ↗︎

messageservice.sender.list - возвращает список зарегистрированных приложением SMS-провайдеров.

Пример

BX24.callMethod(
	'messageservice.sender.list',
	{},
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
			alert("Успешно: " + result.data().join(', '));
	}
);
