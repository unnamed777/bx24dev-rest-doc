catalog.productImage.list ↗︎

catalog.productImage.list(productId, select)

Метод получает список изображений по конкретному товару или торговому предложению.

Параметры

ПараметрТип Описание
productIdstring Идентификатор товара или торгового предложения. Обязательный параметр.
selectobject Список полей для показа в ответе.
startstring Номер страницы вывода. Работает для https запросов.


Примеры

Для JS

BX24.callMethod(
	'catalog.productImage.list',
	{
		productId: 1
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
		result.next();
	}
);

Для HTTPS

https://ваш_портал/rest/catalog.productImage.list?auth=_ключ_авторизации_&start=50
