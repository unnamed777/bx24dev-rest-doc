documentgenerator.document.list ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.document.list(select = ['*'], order = [], filter = [], start = 0)

Возвращает список документов по фильтру.

Параметры

Параметр Описание
selectмассив полей для вывода.
orderмассив для указания порядка вывода {"id": "desc"}.
filterмассив для фильтрации.
startoffset для постраничной навигации.


Пример фильтра

"filter": {
	"value": 5
}

В результате будет список документов, сформированный для внешнего идентификатора = 5.
