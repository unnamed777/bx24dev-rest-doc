crm.measure.fields ↗︎

crm.measure.fields()

Возвращает описание полей для единиц измерений.

Параметры

Без параметров.

Пример

BX24.callMethod(
	"crm.measure.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
