catalog.priceTypeGroup.add ↗︎

catalog.priceTypeGroup.add(fields)

Метод добавляет привязку типа цен к группе покупателей.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.priceTypeGroup.add', 
	{
		fields: {
			catalogGroupId: 14,
			groupId: 16,
			access: "Y"
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
