catalog.price.add ↗︎

catalog.price.add(fields)

Метод добавляет цену товара.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.price.add', 
	{
		fields: {
			catalogGroupId: 1,
			currency: "RUB",
			price: 2000,
			productId: 1
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
