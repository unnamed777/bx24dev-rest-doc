sale.propertygroup.get ↗︎

Scope: sale Права на выполнение: для всех 

sale.propertygroup.get(propertyGroupsId)

Метод для доступа к полям группы свойств.

Если операция успешна, возвращается ресурс группы свойств в теле ответа.

Параметры

ПараметрТип Описание
propertyGroupsIdstring Номер группы свойств.


Примеры

BX24.callMethod(
	'sale.propertygroup.get', 
	{ propertyGroupsId: id }, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
