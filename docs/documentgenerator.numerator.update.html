documentgenerator.numerator.update ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.numerator.update(id, fields)

Метод обновляет существующий нумератор с новыми значениями.

Обновить можно только те нумераторы, которые были созданы через documentgenerator.numerator.add().

Возвращает результат, идентичный documentgenerator.numerator.get().

Параметры

Параметр Описание
idID нумератора
ID нумератораМассив, аналогичный documentgenerator.numerator.add(), только все поля необязательны.
