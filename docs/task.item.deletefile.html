task.item.deletefile ↗︎

Scope: task Права на выполнение: для всех

Метод удаляет привязку файла к задаче.

Параметры метода

ПараметрОписание
authТокен авторизации.
TASK_IDИдентификатор задачи.
ATTACHMENT_IDИдентификатор прикрепленного файла.

Пример вызова

$request = 'http://your-domain.ru/rest/task.item.deletefile.xml?auth=1iqeuq94vzfxu01bouws3voja2lsezfq&TASK_ID=3&ATTACHMENT_ID=28'
