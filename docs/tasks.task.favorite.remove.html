tasks.task.favorite.remove ↗︎

Scope: task Права на выполнение: для всех 

tasks.task.favorite.remove(taskId)

Метод удаления задачи из "Избранного".

В случае успешного выполнения возвращает параметр true (иначе false).

Параметры

Параметр Описание
taskId Идентификатор задачи.

Примеры

BX24.callMethod('tasks.task.favorite.remove', {taskId: 119}, (res)=>{console.log(res.answer.result);});
