sale.shipmentitem.add ↗︎

Scope: sale Права на выполнение: для всех 

sale.shipmentitem.add(fields.shipmentitem)

Метод добавляет элемент коллекции табличной части отгрузки.

Если операция успешна, возвращается ресурс табличной части отгрузки в теле ответа.

Параметры

ПараметрТип Описание
fields.shipmentitemobject Поля, соответствующие доступному списку полей fields.


Примеры

BX24.callMethod(
	'sale.shipmentitem.add', 
	{
		fields: {
			orderDeliveryId: 33,
			basketId: 18,
			quantity: 1
		}
	}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
