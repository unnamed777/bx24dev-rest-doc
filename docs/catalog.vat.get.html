catalog.vat.get ↗︎

catalog.vat.get(id)

Метод для доступа к значениям полей ставки НДС по ID.

Если операция успешна, возвращается ресурс ставки НДС в теле ответа.

Параметры

ПараметрТип Описание
idinteger Идентификатор ставки НДС.

Примеры

BX24.callMethod(
	'catalog.vat.get',
	{
		id: 7
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
