disk.storage.gettypes ↗︎

Scope: disk Права на выполнение: для всех 

disk.storage.gettypes

Возвращает список типов хранилищ.

Пример ответа:

"result": [
	"user", //хранилище пользователей
	"common", //хранилище общих документов
	"group" //хранилище социальных групп
]

Параметры

Без параметров.

Пример

BX24.callMethod(
	"disk.storage.gettypes",
	{},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
