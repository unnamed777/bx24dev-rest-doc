feature.get ↗︎

$result[] = CRest::call(
	'feature.get',
	[
		'CODE' => 'rest_offline_extended',
	]
);

Возвращает информацию о доступности функционала на конкретном портале.

Параметры

Параметр Тип Описание
CODE string Доступные ключи:
  • rest_offline_extended - доступность оффлайн событий
  • rest_auth_connector — доступность ключа auth_connector в события

Ответ и возвращаемое значение

Пример ответа:

'result' =>
	[
		'value' => 'Y'
	]

Возвращаемое значение:

  • value Y - наличие функционала на портале. Y - есть на текущем портале, N - нет на портале
  • lang_selfhosted, где lang заменяется на en, ru, ua, kz и т.д. (для коробок Битрикс24).
