catalog.priceTypeGroup.getFields()

Метод возвращает поля привязки типов цен к группам покупателей.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.priceTypeGroup.getFields', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);

Возвращаемые поля

Поле ОписаниеТипПримечание
access Тип добавляемой привязки:
  • N – право на просмотр этого типа цен;
  • Y – право на покупку по этому типу цен.

Чтобы добавить оба права, необходимо вызвать метод дважды, поочерёдно указав оба типа привязки (N и Y). 		charОбязательное поле.
catalogGroupId ID типа цены.integerОбязательное поле.
groupId ID группы, к которой привязывается цена.integerОбязательное поле.
id Идентификатор привязки.integerНеизменяемое поле.
