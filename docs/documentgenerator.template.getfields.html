documentgenerator.template.getfields ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.template.getfields(id, entityTypeId, entityId, values = [])

На вход получает идентификатор шаблона и возвращает набор полей шаблона с их описанием.

Параметры

Параметр Описание
idID шаблона

Пример

"templateFields": {
	"DocumentNumber": {
		"title": "Номер" // заголовок
		"value": "22" // значение
		"group": [ // иерархия
			0: "Документ"
		],	
		"default": "22" // значение по умолчанию
	},
	"SomeDate": {
		"value": "", 
		"default": ""
	},
	"SomeName": {
		"value": "", 
		"default": ""
	}
}
