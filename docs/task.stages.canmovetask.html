task.stages.canmovetask ↗︎

Scope: task Права на выполнение: для всех

Определяет, может ли текущий пользователь перемещать задачи в указанной сущности.

Параметры

Метод Описание С версии
entityId ID сущности
entityType Тип сущности (U - пользователь, G - группа). В случае U (Мой план) true вернется только в одном случае, если в entityId передаётся идентификатор текущего пользователя.

Возвращает true/false.
