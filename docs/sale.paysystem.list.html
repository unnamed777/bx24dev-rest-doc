sale.paysystem.list ↗︎

sale.paysystem.list(params)

Метод для получения списка платежных систем.

Параметры

ПараметрОписание
params Массив параметров Список доступных полей идентичен полям метода sale.paysystem.add для выборки (select, order, filter) (необязательный).

BX24.callMethod(

	function(result)
