catalog.store.getFields ↗︎

catalog.store.getFields()

Метод возвращает поля складов.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.store.getFields',
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
