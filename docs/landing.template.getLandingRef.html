landing.template.getLandingRef ↗︎

landing.template.getLandingRef(id)

Метод получает список включаемых областей для страницы. Ключами возвращаемого массива являются идентификаторы включаемых областей, а значениями - идентификаторы страниц.

Параметры

Метод Описание
ID Идентификатор страницы

Пример

BX24.callMethod(
	'landing.template.getLandingRef',
	{
		id: 557
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
