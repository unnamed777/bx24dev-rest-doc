catalog.productPropertyFeature.add ↗︎

catalog.productPropertyFeature.add(fields)

Метод добавляет параметр свойства товаров или торговых предложений.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
		'catalog.productPropertyFeature.add', 
	{
		fields: {
			propertyId: 128,
			featureId: "IN_BASKET",
			moduleId: "catalog",
			isEnabled: "Y"
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
