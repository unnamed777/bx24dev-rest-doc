crm.dealcategory.fields

Возвращает описание полей направления сделок.

Параметры

Без параметров

Пример

BX24.callMethod(
	"crm.dealcategory.fields",
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
