crm.deal.delete ↗︎

crm.deal.delete(id)

Удаляет сделку и все связанные с ней объекты: дела, история, дела Таймлайна и другие. Объекты удаляются, если они не привязаны к другим сущностям/элементам. А если привязаны, то удалится только привязка к удаляемой сделке.

Параметры

Параметр Описание
id Идентификатор сделки.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.deal.delete",
	{ id: id },
	function(result)
	{
		if(result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
