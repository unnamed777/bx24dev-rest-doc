documentgenerator.numerator.get ↗︎

Scope: documentgenerator Права на выполнение: для всех 

documentgenerator.numerator.get(id)

Метод возвращает информацию о нумераторе по его идентификатору.

Параметры

Параметр Описание
idID нумератора.


Ответ

"numerator": {
	"id": "202", // id шаблона
	"name": "Rest Template", // название
	"template": "{NUMBER}", // шаблон
	"settings": { // настройки генераторов
		"Bitrix_Main_Numerator_Generator_SequentNumberGenerator": {
			"start":  20,
			"step":  5,
			"periodicBy": '',  
			"timezone": '',  
			"isDirectNumeration": ''  
		}	
	},
}
