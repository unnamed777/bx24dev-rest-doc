crm.measure.delete ↗︎

crm.measure.delete(id)

Удаляет единицу измерения.

Параметры

Параметр Описание
id Идентификатор единицы измерения.

Пример

var id = prompt("Введите ID");
BX24.callMethod(
	"crm.measure.delete",
	{id: id},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.info(result.data());
	}
);
