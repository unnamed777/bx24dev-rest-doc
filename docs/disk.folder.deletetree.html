disk.folder.deletetree ↗︎

Scope: disk Права на выполнение: для всех 

disk.folder.deletetree

Уничтожает папку и всё её дочерние элементы навсегда.

В ответе "result": true - успешное уничтожение папки.

Параметры

Параметр Описание
id Идентификатор папки.

Пример

BX24.callMethod(
	"disk.folder.deletetree",
	{
		id: 8
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
