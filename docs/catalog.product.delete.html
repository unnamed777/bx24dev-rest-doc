catalog.product.delete ↗︎

catalog.product.delete(id)

Метод удаляет товар торгового каталога.

Если операция успешна, возвращается true в теле ответа.

Параметры

ПараметрТип Описание
idstring Номер товара торгового каталога.

Примеры

BX24.callMethod(
	'catalog.product.delete', 
	{
		id: 14
	},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
