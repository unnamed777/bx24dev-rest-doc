bizproc.provider.add ↗︎

Scope: bizproc Права на выполнение: для всех

bizproc.provider.add - регистрирует новый SMS-провайдер в бизнес-процессе.

Внимание! Методы устарели, рекомендуется к использованию Служба сообщений

Параметры

Параметр Описание
CODE Внутренний идентификатор провайдера.
Допустимые символы a-z, A-Z, 0-9, точка, дефис и нижнее подчеркивание.
TYPE Тип провайдера.
HANDLER URL приложения, на который будут отправлены данные.
NAME Название провайдера. Может быть строкой или ассоциативным массивом локализированных строк.
DESCRIPTION Описание провайдера. Может быть строкой или ассоциативным массивом локализированных строк.

Пример

var params = {
	CODE: 'provider1',
	TYPE: 'SMS',
	HANDLER: 'http:///',
	NAME: 'Провайдер SMS.ru',
	DESCRIPTION: 'Провайдер SMS.ru'
};
BX24.callMethod(
	'bizproc.provider.add',
	params,
	function(result)
	{
		if(result.error())
			alert("Error: " + result.error());
		else
			alert("Успешно: " + result.data());
	}
);
