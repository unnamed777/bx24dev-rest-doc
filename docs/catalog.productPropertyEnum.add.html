catalog.productPropertyEnum.add ↗︎

catalog.productPropertyEnum.add(fields)

Метод добавляет значение списочных свойств.

Параметры

ПараметрТип Описание
fieldsobject Поля, соответствующие доступному списку полей fields.

Примеры

BX24.callMethod(
	'catalog.productPropertyEnum.add',
	{
		fields: {
			propertyId: 128,
			value: "Средний",
			def: "Y",
			sort: 123,
			xmlId: "M"
		}
	},
	function(result) {
		if (result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
