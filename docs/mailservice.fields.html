mailservice.fields ↗︎

mailservice.fields(
)

Возвращает описание полей почтового сервиса.

Параметры

Без параметров

Пример

BX24.callMethod(
	"mailservice.fields",
	{
	},
	function(result)
	{
		if(result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.info(result.data());
		}
	}
);
