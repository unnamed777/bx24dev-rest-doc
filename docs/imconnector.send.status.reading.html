imconnector.send.status.reading ↗︎

Метод подтверждения прочтения сообщения из ОЛ во внешнюю систему.

Параметры

Параметр Описание С версии
CONNECTOR ID коннектора (который был указан при регистрации обработчика).
LINE ID открытой линии.
MESSAGES Массив сообщений, где сообщения описываются массивом следующего формата: 
array(
	array(
		'im',//Пересылается элемент 'im' из входящего сообщения ОЛ
		'message' => array(
			'id'//Массив ID во внешней системе. Именно массив, а не единичное 
				//значение, даже если ID один.
		),
		'chat' => array(
			'id'//ID чата во внешней системе.
		),
	),
	array(...),
);
