crm.product.property.list ↗︎

crm.product.property.list(order, filter)

Возвращает список свойств товаров.

Параметры

Параметр Описание
order Поля сортировки.
filter Поля фильтра.

Пример

BX24.callMethod(
	"crm.product.property.list", {
		order: {"SORT": "ASC"},
		filter: {
			"PROPERTY_TYPE": "S",
			"USER_TYPE": "HTML"
		}
	},
	function (result)
	{
		if (result.error())
		{
			console.error(result.error());
		}
		else
		{
			console.dir(result.data());
			if (result.more())
				result.next();
		}
	}
);
