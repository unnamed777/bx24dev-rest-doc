events ↗︎

BX24.callMethod('events', {});

Показ общего списка событий.

scope=имя_разрешения - показ событий, входящих в данное разрешение. Если указан параметр без значения (events?scope=&auth=xxxxx), то будут выведены все общие события.

Пример:

https://my.bitrix24.ru/rest/events?auth=d161f25928c3184678924ec127edd29a - показать все доступные 
события.
