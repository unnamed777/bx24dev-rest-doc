disk.file.getExternalLink ↗︎

Scope: disk Права на выполнение: для всех 

disk.file.getExternalLink(
	id
)

Метод возвращает публичную ссылку по идентификатору файла. Публичные ссылки отдают файл на скачивание только, если пользователь зашел в карточку публичной ссылки.

Пример ответа:

"result": "https://test.bitrix24.ru/~Fjruf2"

Параметры

Параметр Описание
id Идентификатор файла.

Пример

BX24.callMethod(
	"disk.file.getExternalLink",
	{
		id: 10
	},
	function (result)
	{
		if (result.error())
			console.error(result.error());
		else
			console.dir(result.data());
	}
);
