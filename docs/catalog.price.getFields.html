catalog.price.getFields ↗︎

catalog.price.getFields()

Метод возвращает поля цены товара.

Параметры

Без параметров.

Примеры

BX24.callMethod(
	'catalog.price.getFields', 
	{},
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
