crm.activity.todo.updateDescription ↗︎

crm.activity.todo.updateDescription(
	{
		ownerTypeId: number,
		ownerId: number,
		id: number,
		value: string
	}
)

Изменить текст дела.

Параметры

Параметр Описание
ownerTypeId Идентификатор типа элемента (справочник доступных типов), к которому относится дело
ownerId Идентификатор элемента, к которому относится дело
id Идентификатор дела
value Новый текст дела

Результат будет содержать id - идентификатор измененного дела.

Параметры

Пример использования

crm.activity.todo.updateDescription?ownerTypeId=2&ownerId=1&id=1&value=перезвонить

Успешный результат:

{
	"result": {
		"id": 243
	}
}
