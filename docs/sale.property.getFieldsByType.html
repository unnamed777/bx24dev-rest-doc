sale.property.getFieldsByType ↗︎

Scope: sale Права на выполнение: для всех

Описание и пример

sale.property.getFieldsByType()

Метод возвращает поля и настройки свойства заказа для определенного типа свойства.

Параметры

Без параметров.

Пример

BX24.callMethod(
	'sale.property.getFieldsByType', 
	{}, 
	function(result)
	{
		if(result.error())
			console.error(result.error().ex);
		else
			console.log(result.data());
	}
);
