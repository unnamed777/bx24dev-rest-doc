crm.deal.contact.items.set ↗︎

crm.deal.contact.items.set(id, items)

Устанавливает набор контактов, связанных с указанной сделкой.

Параметры

Параметр Описание
idИдентификатор сделки.
itemsНабор контактов в виде массива объектов со следующими полями:
  • CONTACT_ID - идентификатор контакта (обязательное поле)
  • SORT - индекс сортировки
  • IS_PRIMARY - флаг первичного контакта
