crm.timeline.note.delete ↗︎

crm.timeline.note.delete(
	{
		ownerTypeId: number,
		ownerId: number,
		itemType: number,
		itemId: number
	}
)

Удалить заметку

Параметр

Параметр Описание
ownerTypeId Идентификатор типа элемента (справочник доступных типов), к которому относится запись.
ownerId Идентификатор элемента, к которому относится запись
itemType Тип записи, из которой нужно удалить заметку:
  1. запись истории
  2. дело
itemId Идентификатор записи, из которой нужно удалить заметку.
Если itemType=1, то это идентификатор записи истории таймлайна,
Если itemType=2, то это идентификатор дела.

Пример

Пример использования

crm.timeline.note.delete?ownerTypeId=1&ownerId=1&itemId=1&itemType=2

Успешный результат:

{
	"result": true
}
