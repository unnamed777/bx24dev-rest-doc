catalog.documentcontractor.list ↗︎

catalog.documentcontractor.list(select, filter, order)

Метод получает список привязок CRM-сущностей (Контакта/Компании) к документам по фильтру. Возвращает список привязок со структурой, описанной в параметре select (если не задан, то выводятся все поля, как в методе getFields).

Параметры

ПараметрТип Описание
selectobject Поля, соответствующие доступному списку полей fields.
filterobject Поля, соответствующие доступному списку полей fields.
orderobject Поля, соответствующие доступному списку полей fields.
startstring Номер страницы вывода. Работает для https запросов.


Примеры

Для JS

BX.rest.callMethod(
	'catalog.documentcontractor.list',
	{
		select: ['id', 'documentId'],
		order: {'id': 'desc'},
		filter:
		{
			'documentId': 6,
			'entityTypeId': 3
		}
	},
	r => console.log(r.data())
	function(result)
	{
		result.next();
	}
);
